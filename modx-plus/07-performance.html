Performance Mode สำหรับแสดงสด

Performance Mode คือหัวใจของ MODX — รวมเสียงหลาย Part ใน Performance เดียว ทั้ง Layer, Split, ควบคุมด้วย Scene และ Super Knob บทนี้อธิบาย ทีละขั้น พร้อมบอกว่าจอแสดงอะไรตลอดทาง

Performance คืออะไร — ความสัมพันธ์ Part / Performance

โครงสร้าง

1 Performance = รวม Part ได้สูงสุด 8 Parts แต่ละ Part มีเสียง (Voice) ของตัวเอง มี Volume, Pan, Note Range, MIDI Channel ของตัวเอง

กด [HOME] → จอแสดงกริด 8 Part ด้านขวา แต่ละ Part มีชื่อเสียง + Level Meter กระพริบตามเสียงที่ดัง

สร้าง Layer — หลายเสียงดังพร้อมกัน

ขั้นที่ 1 — กด [EDIT]

กด [EDIT]

จอแสดงหน้า Performance Edit: Tab "Common/Audio" ไฮไลต์ ด้านซ้ายมีรายการ Part 1–8 + Common ด้านล่าง
ขั้นที่ 2 — เลือก Part 1 ใส่เสียง Piano
จิ้ม Part 1 ในรายการซ้าย → Tab "Part Settings" เปิด → จิ้มปุ่ม [Category Search] หรือจิ้มชื่อเสียงเดิม
หน้า Category Search: รายการหมวดหมู่เป็น Grid — Piano, Organ, Strings, ฯลฯ
จิ้ม "Piano" → รายการเสียงด้านขวาปรากฏ → จิ้ม "CFX Concert" → จิ้ม [OK]
Part 1 ชื่อเปลี่ยนเป็น "CFX Concert" + Level Meter ของ Part 1 ขยับตามคีย์ที่กด
ขั้นที่ 3 — เพิ่ม Part 2 ใส่เสียง Strings (Layer)
จิ้ม Part 2 ในรายการซ้าย → จิ้มสวิตช์ "Part On/Off" ให้เป็น ON (เปลี่ยนจากสีเทาเป็นสว่าง)
จิ้มชื่อเสียง Part 2 → Category Search เปิด → จิ้ม "Strings" → เลือก "Mellow Strings" → [OK]
Part 2 ชื่อเปลี่ยนเป็น "Mellow Strings" — ตอนนี้กดคีย์จะได้ยิน Piano + Strings พร้อมกัน = Layer สำเร็จ
ขั้นที่ 4 — ปรับ Volume / Pan ของแต่ละ Part
จิ้มค่า Volume ของ Part 2 ในจอ → หมุน Data Dial ปรับ Strings ให้เบากว่า Piano (เช่น 80 แทน 127)
จิ้มค่า Pan → หมุน Dial ให้ Strings อยู่ขวาเล็กน้อย (Pan = R10)

สร้าง Split — เสียงต่างกันในช่วงคีย์ต่าง

ขั้นที่ 1-3 เหมือน Layer แต่เพิ่ม Part 2 ด้วยเสียง Bass
Part 1 = Piano, Part 2 = Bass → ตอนนี้ยังดัง Layer กัน → ต้องกำหนด Note Range
ขั้นที่ 2 — ตั้ง Note Range ของ Part 1 (ทำนอง)
จิ้ม Part 1 → จิ้ม Tab "Zone/Key" (หรือ Part Settings > Note Limit)
จอแสดง Piano Roll แนวนอนยาวตลอดจอ มีแถบสีไฮไลต์ช่วงโน้ตที่ Part นี้ Response (ค่า Default = C-2 ถึง G8)
จิ้มที่ขอบซ้ายของแถบแล้วลาก → หรือจิ้มช่อง "Note Low" → หมุน Data Dial ตั้งเป็น E3 (กลางของ Piano)
แถบ Part 1 หดเหลือแค่ E3–G8 = Piano อยู่แค่ช่วงบน
ขั้นที่ 3 — ตั้ง Note Range ของ Part 2 (Bass)
จิ้ม Part 2 → Tab "Zone/Key" → ตั้ง "Note High" เป็น D#3
แถบ Part 2 อยู่ C-2–D#3 = Bass อยู่แค่ช่วงล่าง — ตอนนี้กดคีย์สูง = Piano เท่านั้น, กดคีย์ต่ำ = Bass เท่านั้น
อย่าลืม Store Performance หลังตั้ง Split เสร็จ ไม่งั้นปิดเครื่องหรือเปลี่ยน Performance ข้อมูลหาย

Scene — เก็บ Snapshot ของระดับ Volume / Knob

บันทึก Scene

กด [HOME] เพื่อกลับหน้าหลัก → ปรับ Volume แต่ละ Part และ Super Knob ให้อยู่ในตำแหน่งต้องการ → กด Scene Button (ปุ่มสี 1–8) ค้าง 1 วินาที

Pop-up เล็กๆ: "Scene 1 Stored" โผล่ขึ้น 1 วินาทีแล้วหายไปเอง — ค่า Volume และตำแหน่ง Super Knob ถูกเก็บใน Scene นั้น
สลับ Scene ระหว่างเล่น

กด Scene Button ที่ต้องการ (1–8) ขณะแสดงสด

ปุ่ม Scene ที่กดสว่างขึ้น / ปุ่มอื่นหรี่ลง + Volume Part และ Super Knob ค่อยๆ Morph ไปยังค่าที่บันทึกใน Scene นั้น (ถ้าเปิด Morph Mode) หรือเปลี่ยนทันที

Super Knob — Assign ให้ Control หลาย Parameter พร้อมกัน

วิธี Assign Super Knob

กด [EDIT] → จิ้ม Tab "Common/Audio" → จิ้ม Tab ย่อย "Control" → จิ้ม "Super Knob"

จอแสดงหน้า Super Knob Assign: รายการ Knob Connection แต่ละ Row มีช่อง "Src" (Super Knob) → "Dest" (Parameter ปลายทาง) → Depth (ความแรง)
จิ้ม [+] เพื่อเพิ่ม Row ใหม่ → จิ้ม "Dest" → Browser เลือก Parameter: เช่น Part1 > Filter Cutoff → OK
จิ้มช่อง "Depth" → หมุน Data Dial ตั้งค่า เช่น +63 = หมุน Super Knob ไปขวาสุด → Cutoff เพิ่มสูงสุด
ทดสอบ

กด [HOME] → หมุน Super Knob

ค่า Cutoff ของ Part 1 เปลี่ยนตาม Super Knob + ถ้า Assign ไว้หลาย Part จะเปลี่ยนพร้อมกันทุก Part ที่ Assign ไว้

Motion Sequencer — Automation อัตโนมัติ

เปิด Motion Sequencer

กด [EDIT] → เลือก Part → จิ้ม Tab "Mod/Control" → จิ้ม "Motion Seq"

จอแสดง Lane 1-4 เป็น Step Sequencer แนวนอน แต่ละ Step มีปุ่มสี่เหลี่ยมที่จิ้มเพื่อเปิด/ปิดหรือลาก
จิ้ม Step ที่ต้องการ Activate → Step สว่างขึ้น = เปิด / จิ้มอีกครั้ง = ปิด
จิ้มค่า "Dest" ของ Lane → เลือก Parameter: เช่น Cutoff, Volume, Pan → กด [PLAY] เพื่อฟัง Motion Seq ทำงาน
LED ที่ Scene Button กะพริบตาม Beat / จอแสดง Step ปัจจุบันไฮไลต์ไล่ไปทีละ Step

บันทึก Performance ที่สร้าง

Store Performance

กด [STORE] → Pop-up "Store Performance" → เลือก Bank + Slot + ตั้งชื่อ → จิ้ม [OK]

ชื่อ Performance บนจอไม่มี "*" แล้ว = บันทึกสำเร็จ
💡 เคล็ดลับ Live Performance

ตั้ง Scene 1 = เสียงเบา (Intro) / Scene 2 = เสียงปกติ (Verse) / Scene 3 = เต็ม (Chorus) / Scene 8 = เสียงดังสุด (Climax) → กดสลับด้วยนิ้วโป้งขณะเล่นได้เลย ไม่ต้องหยุดมองจอ
