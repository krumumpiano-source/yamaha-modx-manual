วิธีแต่งเสียง AWM2 & FM-X

การแต่งเสียงบน MODX แบ่งเป็น 2 เส้นทางหลัก: AWM2 (Sample) สำหรับเสียง Realistic และ FM-X สำหรับเสียง Electric/Digital/Metallic — บทนี้จะพาทำทีละขั้น

แต่งเสียง AWM2 — Step by Step

ขั้นที่ 1

เลือก Performance ที่ต้องการแก้ไข → กด [EDIT] → เลือก Part ที่ใช้ AWM2 → กด Part Edit

ขั้นที่ 2

Tab OSC/Tune → เลือก Waveform → ฟัง Preview → กำหนด Note Range ถ้าต้องการให้ Sound นี้ทำงานแค่บางส่วนของคีย์

ขั้นที่ 3

Tab Filter → ปรับ Cutoff (ลดก็ทำให้เสียง Muffled/Warm เพิ่มก็ทำให้ Bright) → ปรับ Resonance (เพิ่มทำให้ได้ยินเสียง "Wah" รอบ Cutoff frequency)

ขั้นที่ 4

Tab Amp → ปรับ Envelope:
- Attack: เสียงขึ้นเร็วหรือช้า (0 = ทันที, 127 = ค่อยขึ้น)
- Decay: ลดลงหลัง Attack
- Sustain: ระดับเสียงตอนกดค้าง
- Release: หางเสียงหลังปล่อยคีย์

ขั้นที่ 5

Tab Effect → เลือก Insertion Effect เช่น Chorus, Phaser, Distortion → บันทึก Store

แต่งเสียง FM-X — Step by Step

ขั้นที่ 1

เลือก Part FM-X → Part Edit → Tab Algorithm → ลอง Algorithm 1 (ง่ายที่สุด: Op1-2-3-4 เรียงตรง) แล้วค่อยลองซับซ้อนขึ้น

ขั้นที่ 2

Tab Operator → เลือก Op1 (เสียงหลัก Carrier) → ปรับ Level ให้ดัง → ปรับ Ratio (เช่น 1.00 = pitch ปกติ, 2.00 = octave ขึ้น)

ขั้นที่ 3

เลือก Op2 (Modulator ของ Op1) → เพิ่ม Level → เสียงจะ "Metallic/Bright" ขึ้น → ลอง Ratio ต่างๆ เช่น 3.5, 7.0 สำหรับเสียง Bell หรือ Gong

ขั้นที่ 4

Tab Envelope → ปรับ Envelope ของแต่ละ Op แยกกัน → Op ที่เป็น Carrier ปรับให้ยาว, Op Modulator ให้ Decay สั้น = เสียงเปลี่ยนจาก Bright ไป Mellow ตาม Time

💡 เคล็ดลับ FM

เริ่มจาก Preset FM Ep (Electric Piano) แล้วเปลี่ยนเล็กน้อย ง่ายกว่าเริ่มจาก Scratch มาก FM EP ที่ดีต้องใช้ Algorithm 5 + Fine Tune Ratio ของ Op ให้ใกล้กัน
