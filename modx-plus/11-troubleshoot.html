แก้ไขปัญหาที่พบบ่อย

รวมปัญหาที่พบบ่อยใน MODX และวิธีแก้ไข step-by-step

ปัญหา: ไม่มีเสียงออก

ตรวจสอบ 1

Master Volume knob (ด้านซ้ายสุด) ต้องไม่อยู่ที่ 0 → หมุนขึ้นอย่างน้อย 50%

ตรวจสอบ 2

Part Volume: กด Part → ตรวจ Volume ใน Part Setting ต้องไม่เป็น 0

ตรวจสอบ 3

สายออก: OUTPUT L+R → ต่อกับ Mixer/Speaker correctly? ลอง Headphone เพื่อแยก Hardware vs Software

ตรวจสอบ 4

Mute: Part นั้น Mute อยู่หรือเปล่า? กด Shift + Part Button เพื่อ Unmute

ปัญหา: MIDI ไม่ทำงานกับ DAW

ตรวจสอบ 1

Utility → MIDI I/O = USB (ไม่ใช่ DIN)

ตรวจสอบ 2

ติดตั้ง Yamaha Steinberg USB Driver แล้วหรือยัง? ถ้ายัง ดาวน์โหลดจาก yamaha.com

ตรวจสอบ 3

DAW: MIDI Track Input = MODX MIDI Port 1 / Local Control = Off ถ้าได้ยินเสียง 2 ครั้ง (Echo)

ปัญหา: เสียงช้า (Latency สูง)

แก้ไข

ลด Buffer Size ใน Audio Interface Driver / ใช้ ASIO Driver (Windows) หรือ Core Audio (Mac) / ปิด WiFi ขณะใช้งาน DAW Real-time

ปัญหา: เครื่องค้าง / Freeze

แก้ไข

กด Power ค้าง 5 วินาทีเพื่อ Force Shutdown → เปิดใหม่ ถ้าเกิดบ่อย → Update Firmware เป็นเวอร์ชันล่าสุด หรือ Factory Reset

ปัญหา: USB Flash Drive ไม่รู้จัก

แก้ไข

Format USB เป็น FAT32 (ไม่ใช่ exFAT หรือ NTFS) / ใช้ USB ความเร็วปกติ (ไม่ต้องใช้ USB 3.0 Speed เร็วกว่าก็ไม่ได้ช่วย)
