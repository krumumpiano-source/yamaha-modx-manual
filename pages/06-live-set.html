บทที่ 06 — Performance

Live Set & Gig

Live Set คือเครื่องมือสำหรับนักดนตรีที่ต้องการจัด Performance หลายชุดไว้ใน Set List เดียว เปลี่ยน Performance ขณะเล่นสดได้อย่างรวดเร็วและแม่นยำ

ทุกรุ่น

Live Set คืออะไร?

Live Set คือ Container ที่เก็บรายชื่อ Performance ไว้ในรูปแบบ Grid 8×16 = 128 Slot ต่อ Live Set และสามารถมี Live Set ได้หลายชุด เก็บไว้ใน User Memory หรือ USB Flash Drive

วิธีสร้าง Live Set

  1. 1
    เข้า Live Set Mode — กดปุ่ม LIVE SET บน Front Panel (หรือแตะ Live Set บน Touchscreen)
  2. 2
    เลือก Live Set ว่าง — หน้าจอแสดง Grid ของ Live Set ที่มีอยู่ เลือก Empty Slot หรือสร้างใหม่
  3. 3
    เปิด Edit Mode — กด EDIT หรือแตะ Edit บน Touchscreen
  4. 4
    เลือก Slot A-1 → กด ASSIGN PERFORMANCE → เลือก Performance ที่ต้องการ เช่น "CFX Full Grand"
  5. 5
    ทำซ้ำสำหรับ Slot A-2, A-3... เรียงตามลำดับเพลงในเซ็ท
  6. 6
    บันทึก Live Set → กด STORE → ตั้งชื่อ เช่น "วง ABC - Concert 2026"

การเปลี่ยน Performance ระหว่างเล่น

วิธีที่ 1: กด Bank + Slot โดยตรง

  1. 1
    กด Bank A
  2. 2
    กด Performance Number เช่น 3 → โหลด Performance A-3 ทันที

⚡ การเปลี่ยนเกิดขึ้นทันที (Instant) — เสียง Sustain ค้างจาก Performance เก่ายังคงอยู่ (Sound Remains)

วิธีที่ 2: Foot Switch เปลี่ยนหน้า/ถอยหลัง

ตั้ง Foot Switch ให้เปลี่ยน Performance หน้าถัดไปได้โดยไม่ต้องแตะ Keyboard โดยเด็ดขาด เหมาะสำหรับแสดงสด:

  1. 1
    กด UTILITY → Settings → Controller → Foot Switch
  2. 2
    ตั้ง Function เป็น Inc Live Set Slot (เพิ่ม Slot ถัดไป)
  3. 3
    กด Foot Switch ขณะเล่น → Performance เปลี่ยนไป Slot ถัดไปอัตโนมัติ

Volume ใน Live Set

แต่ละ Slot ใน Live Set สามารถตั้ง Volume Offset ได้ (-12dB ถึง +12dB) ช่วยให้ Performance ที่อาจมี Volume ต่างกันออกมาสม่ำเสมอ:

  1. 1
    เข้า Live Set Edit → เลือก Slot ที่ต้องการ
  2. 2
    ปรับ Level เพิ่ม/ลด เพื่อ Balance กับ Slot อื่น
  3. 3
    ทดสอบโดยกดสลับระหว่าง Slot ฟัง Volume Consistency

บันทึกและโหลด Live Set

บันทึกไปยัง USB Flash Drive

  1. 1
    เสียบ USB Flash Drive เข้าช่อง USB TO DEVICE
  2. 2
    กด UTILITY → Storage → Save → เลือก Live Set
  3. 3
    ตั้งชื่อไฟล์ → กด STORE → ไฟล์ .X8L ถูกบันทึก

โหลด Live Set

  1. 1
    เสียบ USB Flash Drive
  2. 2
    กด UTILITY → Storage → Load → เลือกไฟล์ .X8L
  3. 3
    Confirm → Live Set โหลดเข้ามา พร้อมใช้งาน
⚠️

Backup ก่อนทุกครั้ง

ก่อนทำการ Update Firmware หรือ Factory Reset ควร Save Live Set และ Performance ทั้งหมดไปยัง USB Flash Drive ก่อนเสมอ เพื่อป้องกันข้อมูลสูญหาย
