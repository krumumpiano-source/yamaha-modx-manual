MIDI Setup

การตั้งค่า MIDI สำหรับการบันทึกเสียงใน DAW การต่อ Controller ภายนอก และการใช้ MODX เป็นส่วนหนึ่งของ Studio Setup หรือวงดนตรี

ทุกรุ่น

การตั้งค่า MIDI พื้นฐาน

เข้าเมนู MIDI Settings: กด UTILITY → Settings → MIDI I/O

MIDI Channel

MODX รับ MIDI ผ่าน 2 ช่องทาง:

  • MIDI IN (DIN 5-pin) — จาก External Controller หรือ Keyboard อื่น
  • USB TO HOST — จาก Computer (DAW, Sequencer)
Settingค่าคำอธิบาย
Common MIDI Channel1-16Channel หลักที่ Keyboard ของ MODX ส่งออก และรับ MIDI สำหรับ Omni Receive
Part MIDI Channel1-16 ต่อ PartChannel ที่ Part นั้นรับ — ตั้งผ่าน Part Edit → MIDI → Receive Ch
MIDI Out PortMIDI, USB, Bothส่ง MIDI ออกทางไหน
ClockInternal, MIDI, USB, AutoClock Source สำหรับ Arp, Motion Seq

Local Control On/Off — สำคัญมากเมื่อใช้กับ DAW

⚠️

Local OFF เมื่อใช้กับ DAW

ถ้าใช้ MODX กับ DAW (Record MIDI ผ่าน DAW แล้วส่งกลับมา) ต้องตั้ง Local Control = OFF
ไม่งั้นจะได้ยินเสียงซ้ำ (Double) เพราะ Keyboard เล่นส่วนหนึ่ง + DAW ส่ง MIDI กลับมาอีกส่วนหนึ่ง

  1. 1
    กด UTILITY → Settings → Advanced → MIDI
  2. 2
    ค้นหา Local Control → เปลี่ยนเป็น Off
  3. 3
    DAW ตั้งค่า MIDI Thru = On เพื่อส่ง MIDI กลับมาที่ MODX
  4. 4
    ตอนนี้เล่น Keyboard → MIDI ออก DAW → MIDI กลับมาขับ Sound ของ MODX (ไม่ Double)

เล่น Live โดยไม่มี DAW → Local Control = On เสมอ

ต่อ Controller ภายนอก (External MIDI Keyboard)

สามารถต่อ MIDI Keyboard ภายนอกเพื่อเพิ่ม Range หรือ Control MODX จาก Keyboard อื่น:

  1. 1
    ผ่าน MIDI DIN: ต่อ MIDI OUT ของ Controller → MIDI IN ของ MODX
  2. 2
    ผ่าน USB: ต่อ USB ของ Controller → USB Hub → MODX USB TO HOST หรือผ่าน Computer
  3. 3
    ตั้ง Part Receive Channel ให้ตรงกับ Channel ที่ Controller ส่งออก เช่น Controller = CH 2, Part 3 = Receive CH 2
  4. 4
    ทดสอบโดยเล่น Controller ภายนอก — ถ้า Part 3 ออกเสียง = ตั้งค่าถูกต้อง

MIDI OUT — ควบคุม External Sound Module

MODX ส่ง MIDI ออกได้ผ่าน MIDI OUT หรือ USB เพื่อควบคุม Sound Module ภายนอก:

  • ต่อ MIDI OUT → Sound Module's MIDI IN
  • ตั้ง Transmit Channel ของ MODX ให้ตรงกับ Channel ของ Module
  • MODX Common Channel = เล่น Keyboard → ส่ง MIDI ออก Channel นั้น

Zone/Layer ด้วย MIDI Channel

สามารถสร้าง Multi-Zone Performance โดยใช้ MIDI Channel:

🎹

ตัวอย่าง 3-Zone Setup สำหรับ Live

Zone 1 (C-2 – B3) → Part 1 CH1 = Bass Sound
Zone 2 (C4 – B5) → Part 2 CH2 = Piano Sound
Zone 3 (C6 – G8) → Part 3 CH3 = Melody Sound
เล่น Keyboard เดียว ได้ 3 เสียงพร้อมกันตาม Range

MIDI Thru

MIDI THRU ส่ง MIDI ที่รับมาจาก MIDI IN ออกไปยังอุปกรณ์ถัดไปโดยไม่ประมวลผล เหมาะสำหรับ Daisy-chain หลายอุปกรณ์:

Computer USB → MODX USB IN → MIDI THRU → Sound Module → MIDI THRU → Drum Machine
✅
