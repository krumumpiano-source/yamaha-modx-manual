1. Insertion Effects — Effect ต่อ Part

แต่ละ Part มี Insertion Effect ได้ 2 ตัว (Ins.A และ Ins.B) ต่อกันแบบ Serial (A → B) มี Effect Type ให้เลือกกว่า 100 ประเภท

วิธีตั้งค่า Insertion Effect

  1. 1
    เข้า Part Edit → เลือก Part ที่ต้องการ → กด Effect
  2. 2
    เลือก Ins. A หรือ Ins. B → ตั้ง Effect Type จากรายการ
  3. 3
    ปรับ Parameter ของ Effect (แตกต่างกันตาม Effect Type)
  4. 4
    ตั้ง Ins. Effect Output → ส่งไปที่ L&R Output หรือ ส่ง Dry Signal เข้า System Effect ด้วย

2. System Effects — Reverb & Variation

System Effects รับ Signal จากทุก Part (ผ่าน Send Level) มาผสมรวมกัน ทำให้ทุก Part อยู่ใน "ห้อง" เดียวกัน

System Reverb

มี 1 Reverb Algorithm สำหรับ Performance ทั้งหมด เลือก Type ได้:

Reverb Typeลักษณะเหมาะกับ
Hall 1Concert Hall ขนาดกลาง สมดุลPiano, Orchestra → แนะนำสำหรับวง
Hall 2Concert Hall ขนาดใหญ่ Reverb ยาวBallad, Emotional Performance
Roomห้องเล็ก Reverb สั้นJazz, Intimate, Studio Sound
StageเวทีการแสดงLive Performance สมจริง
PlateMetal Plate Reverb คลาสสิกDrum, Snare, Vocal
SpringSpring Tank จาก AmpGuitar, Surf, Vintage
White RoomBright, AiryPop, Synth, Pad

Reverb Parameters ที่สำคัญ

  • Reverb Time — ความยาว Reverb Tail (วินาที) แนะนำสำหรับวง: 1.5-2.5 วินาที
  • Diffusion — ความกระจาย เพิ่มขึ้น = ฟัง Smooth ขึ้น
  • Pre-Delay — หน่วงก่อน Reverb เริ่ม (ms) แนะนำ: Piano 20-30ms
  • High Damp — ลด High Frequency ใน Reverb Tail = เสียงอุ่น ไม่แหลม
  • Return Level — Volume ของ Reverb รวม ก่อน Master

Reverb Send per Part

แต่ละ Part ส่ง Signal ไป Reverb ด้วย Send Level ต่างกัน:

Part / เสียงReverb Send แนะนำเหตุผล
Piano หลัก25-40ต้องการ Space แต่ไม่หนืด
String Pad50-70Pad ชอบ Reverb มากๆ = Ambient
Bass0-10Bass ต้องแน่น Reverb น้อย
Organ15-30Organ มี Rotary แล้ว Reverb ไม่ต้องมาก
Brass20-35Brass ตาม Natural Reverb Hall
Synth Lead10-25Lead ต้องชัด ไม่ Washy

System Variation Effect

Variation เป็น Effect ที่สองใน System สามารถเป็น Delay, Chorus, Flanger, Tremolo, Phaser ฯลฯ ใช้ได้กับ Part ที่ต้องการเอฟเฟกต์พิเศษ:

  • Stereo Delay: เสียงสะท้อนซ้าย-ขวา เหมาะ Guitar, Lead
  • Modulation Delay: Delay + Chorus รวม
  • Tremolo: Volume สั่น เหมาะ Organ, Guitar

3. Master Effects

Master Effects ใส่หลัง Mix ทั้งหมด มี 2 Slots เลือก Algorithm ได้เหมือน Insertion Effect:

  • Master Compressor/Limiter: กัน Clip รักษา Level โดยรวม แนะนำ Ratio 2:1-4:1, Threshold ปรับจน GR ≈ 2-3dB ขณะเล่น Loud
  • Stereo Enhancer: ขยาย Stereo Width โดยรวม
  • Multi-Band Compressor: ควบคุม Dynamic ต่อย่านความถี่

4. Master EQ (5-Band)

Master EQ อยู่สุดท้ายก่อนออก Output ปรับ Tone โดยรวมของทั้ง Performance:

BandFrequencyการใช้งาน
LowLow Shelf ~80Hzเพิ่ม/ลด Bass โดยรวม
Low-Mid250-500Hz Parametricแก้ Muddiness (เสียงขุ่น)
Mid1-3kHz ParametricPresence, Clarity
High-Mid3-8kHz ParametricAir, Detail
HighHigh Shelf ~12kHzเพิ่ม Brilliance, Air
🎚️

EQ Setting สำหรับวงดนตรีสากล

Low Shelf 80Hz: -2dB (ลด Rumble จาก Stage)
Low-Mid 300Hz: -2dB (แก้ Muddiness ที่พบบ่อยในห้องซ้อม)
Mid 2kHz: 0 (ปล่อยช่วงนี้ตาม Source)
High Shelf 10kHz: +1.5dB (เพิ่ม Air, ความสดใสโดยรวม)
