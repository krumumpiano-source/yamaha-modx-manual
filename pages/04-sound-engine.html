AWM2 + FM-X Sound Engine

เข้าใจหัวใจของ MODX — สองเทคโนโลยีที่ทำงานร่วมกันเพื่อสร้างเสียงที่กว้างที่สุด ตั้งแต่ Grand Piano สมจริงไปจนถึง FM Synth ที่ไม่มีใครเทียบ

ทุกรุ่น AN-X เฉพาะ MODXM

ZG Sound Generator — สถาปัตยกรรม

ZG (Zero Glide) Sound Generator คือ Processing Engine หลักของ MODX รองรับการสร้างเสียงได้พร้อมกันหลายรูปแบบใน Performance เดียว:

Performance (1 ชุด)
├── Part 1  → AWM2 (เช่น Piano)
├── Part 2  → AWM2 (เช่น String Pad)
├── Part 3  → FM-X (เช่น DX Bass)
├── Part 4  → FM-X (เช่น FM Bell)
├── Part 5  → AWM2 (เช่น Choir)
├── Part 6  → AWM2 (เช่น Brass)
├── Part 7  → AN-X (MODXM only — เช่น Analog Lead)
└── Part 8  → AWM2 (เช่น Drum Kit)

ทุก Part ออกมารวมที่ Mixer จากนั้นผ่าน System Effects → Master Effects → Output

AWM2 — Advanced Wave Memory 2

AWM2 คือ Sample-based Synthesis ที่ใช้ Wave (ไฟล์เสียงจริงที่บันทึกมา) เป็นฐาน Yamaha บันทึก Sample จาก Musical Instrument จริง เช่น Yamaha CFX Concert Grand Piano, Bösendorfer Imperial, และเครื่องดนตรีออร์เคสตราต่างๆ

โครงสร้าง AWM2 Part

AWM2 Part
└── Element 1-8 (AWM2 มี Element ได้สูงสุด 8 ต่อ Part)
    ├── Oscillator (เลือก Wave จาก ROM)
    │   ├── Wave Select (เลือกชื่อ Wave)
    │   ├── Coarse Tune, Fine Tune
    │   ├── Pan, Volume
    │   └── Velocity Curve
    ├── Filter (กรองความถี่)
    │   ├── Type: LPF (Low Pass), HPF (High Pass), BPF, etc.
    │   ├── Cutoff Frequency (0-127)
    │   ├── Resonance (0-127)
    │   ├── Filter EG (Envelope สำหรับ Filter)
    │   └── Velocity → Cutoff Sensitivity
    ├── Amplifier (ปรับระดับเสียง)
    │   ├── Level (0-127)
    │   ├── Pan (-64 to +63)
    │   └── Amp EG: Attack, Decay, Sustain, Release
    ├── LFO (Low Frequency Oscillator — ความสั่น)
    │   ├── Wave: Sine, Triangle, Sawtooth, Square, S/H
    │   ├── Speed (ความเร็ว)
    │   ├── Depth (ความลึก)

    └── Pitch EG (Pitch Envelope)
        └── ควบคุม Pitch การเปลี่ยนแปลงตามเวลา

เทคนิค Multi-Sampling ของ AWM2

Yamaha บันทึก Sample แต่ละโน้ตบน Piano ทุก 2-3 คีย์ และบันทึกหลาย Velocity Layer (เบา/กลาง/แรง) เพื่อให้เสียงสมจริงสูงสุด กดเบาได้ยิน Sample เบา กดแรงได้ยิน Sample แรงที่แตกต่างกันจริง ไม่ใช่แค่ Volume ต่างกัน

Filter ที่ควรรู้จัก

Filter Typeคุณสมบัติใช้กับ
LPF (Low Pass Filter)ปล่อยความถี่ต่ำ กรองความถี่สูง → เสียงมืด/นุ่มPad, Bass, Mellow Strings
HPF (High Pass Filter)ปล่อยความถี่สูง กรองความถี่ต่ำ → เสียงบาง/สว่างWind, Hi-Hat, Treble Boost
BPF (Band Pass Filter)ปล่อยเฉพาะความถี่กลาง → เสียงเหมือนพูดผ่านโทรศัพท์Wah Effect, Nasal Tone
LPF+HPFNotch / Band เลือกได้ละเอียดมากComplex tone shaping

FM-X — 8-Operator FM Synthesis

FM-X คือ Frequency Modulation Synthesis วิธีสร้างเสียงโดยใช้ Oscillator หนึ่ง (Modulator) เปลี่ยนความถี่ของอีก Oscillator หนึ่ง (Carrier) ได้เสียงที่ไม่สามารถสร้างจาก Sample ได้ เช่น เสียงโลหะ, Bell, E-Piano โลหะ

Operator คืออะไร?

MODX FM-X มี 8 Operators (เทียบกับ DX7 ที่มี 6) แต่ละ Operator คือ Sine Wave Oscillator ที่มี:

  • Frequency Ratio — ความถี่ของ Operator นี้เทียบกับ Note ที่เล่น (เช่น 1.0, 2.0, 0.5)
  • Output Level — ความดังของ Operator (0-127) ถ้าเป็น Carrier นี่คือ Volume, ถ้าเป็น Modulator นี่คือ Modulation Depth
  • EG (Envelope Generator) 7-Stage — กำหนดรูปแบบ Volume เปลี่ยนตามเวลา (ละเอียดกว่า ADSR ปกติ)
  • AM Sensitivity — ความไวต่อ Amplitude Modulation
  • Feedback — Operator ส่ง Signal กลับหาตัวเอง สร้างเสียง Noisy/Aggressive

Algorithm คืออะไร?

Algorithm กำหนดว่า Operator ไหนเป็น Carrier (ออกเสียง) และ Operator ไหนเป็น Modulator (ปรับเปลี่ยนเสียง) MODX FM-X มี 88 Algorithm

🔬

ตัวอย่าง Algorithm ง่ายที่สุด

Algorithm 1: Op.8 → Op.7 → Op.6 → Op.5→Op.4→Op.3→Op.2→Op.1 (Carrier) — Modulator ซ้อนกัน ได้เสียงซับซ้อนมาก
Algorithm 88: Op.1-8 ทุกตัวเป็น Carrier ทั้งหมด — เสียงง่าย แต่ได้ Timbre หลากหลายพร้อมกัน

AN-X — Analog Physical Modeling (เฉพาะ MODXM)

🟣

เฉพาะ MODXM เท่านั้น

AN-X (Analog Physical Modeling Synthesis) เป็นเทคโนโลยีใหม่ที่มาพร้อม MODXM จำลองการทำงานของ Analog Synthesizer แบบ Physical Model ไม่ใช่ Sample ได้เสียง Analog ที่ "มีชีวิต" เปลี่ยนแปลงตลอดเวลาเหมือน Analog จริงๆ
