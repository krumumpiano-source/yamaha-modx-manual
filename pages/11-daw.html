DAW Integration

ใช้ MODX+ และ MODXM เป็น USB Audio Interface ในตัว บันทึกเสียงหลายแทร็คพร้อมกัน ทำงานร่วมกับ Cubase, Logic, Ableton และ DAW อื่นๆ

MODX+ MODXM
⚠️

MODX รุ่นแรก (ไม่มีเครื่องหมาย + และ M)

MODX รุ่นแรก ไม่มี USB Audio Interface ในตัว สามารถส่ง MIDI ผ่าน USB ได้ แต่ต้องการ External Audio Interface สำหรับบันทึกเสียง เชื่อมต่อ Output L/R → Audio Interface ของคุณ

การติดตั้ง Driver

MODX+ และ MODXM ใช้ Yamaha Steinberg USB Driver (Windows และ macOS):

  1. 1
    ไปที่ usa.yamaha.com/products/musical-instruments/keyboards/synthesizers/modx → Downloads
  2. 2
    ดาวน์โหลด Yamaha Steinberg USB Driver ตาม OS ของคุณ
  3. 3
    ติดตั้ง Driver ก่อนเสียบ USB Cable
  4. 4
    Restart Computer → เสียบ USB → ตรวจสอบว่า "Yamaha MODX+" หรือ "MODXM" ปรากฏใน Device List
  5. 5
    ตั้ง Buffer Size ใน Driver: 256 samples สำหรับ Live Play, 512-1024 สำหรับ Recording เพื่อลด CPU Load

ตั้งค่าใน MODX+ / MODXM

  1. 1
    กด UTILITY → Settings → Advanced → USB I/O
  2. 2
    ตั้ง USB I/O Mode = On
  3. 3
    เลือก Sample Rate: 44100 Hz (คนทั่วไป) หรือ 48000 Hz (Video Production)
  4. 4
    ตั้ง Direct Monitor = Off ถ้าต้องการ Monitor ผ่าน DAW เท่านั้น

วิธีใช้งานกับ Cubase (แนะนำ)

Yamaha เป็นเจ้าของ Steinberg ผู้ผลิต Cubase — MODX+ และ Cubase ทำงานร่วมกันได้ดีที่สุด:

  1. 1
    เปิด Cubase → Studio Setup → VST Audio System → เลือก Yamaha Steinberg USB ASIO
  2. 2
    Audio Connections (F4) → Inputs → Add Bus → กำหนด Input 1-2 = MODX+ Ch1-2
  3. 3
    สร้าง Audio Track → Input = MODX+ → กด Record → เล่น Keyboard → บันทึกได้เลย
  4. 4
    สำหรับ Multi-track: Add Bus สำหรับแต่ละ Channel ที่ต้องการแยก (CH3-4, CH5-6 ฯลฯ)

วิธีใช้งานกับ Logic Pro (Mac)

  1. 1
    Preferences → Audio → Output Device และ Input Device → เลือก MODXM หรือ MODX+
  2. 2
    สร้าง Audio Track → Input 1-2 → Record Enable → เล่น
  3. 3
    ตั้ง I/O Buffer Size: Logic → Preferences → Audio → I/O Buffer Size (256 samples สำหรับ Low Latency)

วิธีใช้งานกับ Ableton Live

  1. 1
    Preferences → Audio → Driver Type = ASIO (Win) หรือ CoreAudio (Mac) → Audio Device = MODX+
  2. 2
    Input Config → เปิด Input ที่ต้องการ (1/2, 3/4 ฯลฯ)
  3. 3
    สร้าง Audio Track → Audio From = Ext. In → เลือก Input → Arm Recording

Recording MIDI + Audio พร้อมกัน

💡

Workflow แนะนำ

1. Record MIDI Track ก่อน (ง่ายต่อการแก้ไข Note ผิด)
2. เมื่อพอใจ → Bounce to Audio Track
3. ปิด MIDI Track → ใช้ Audio Track ต่อ
4. ประหยัด CPU และ RAM ได้มาก

ตั้ง DAW เป็น MIDI Clock Master

เพื่อให้ Arpeggio และ Motion Sequencer ของ MODX+ Sync กับ DAW:

  1. 1
    ใน DAW: Project Settings → MIDI Clock Output = On → Port = MODX+
  2. 2
    ใน MODX+: UTILITY → Settings → Advanced → Clock = USB
  3. 3
    กด Play ใน DAW → Arpeggio ของ MODX+ จะ Sync ตาม BPM ของ DAW

A/D Input (Mic / Guitar)

MODX+ และ MODXM มี A/D Input (6.3mm / XLR) เพื่อ Route เสียง Microphone หรือ Guitar เข้า Signal Chain ของ MODX แล้วออก USB:

  • เชื่อมต่อ Mic → A/D Input → ผ่าน Effects ของ MODX → ออก USB เข้า DAW
  • ตั้งค่า: UTILITY → Settings → Audio → A/D Input
  • ปรับ Gain, Low Cut, Phase ได้จาก MODX
