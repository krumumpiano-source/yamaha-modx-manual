บทที่ 05 — Performance

Performance Mode

Performance คือหน่วยหลักในการเล่น MODX — ทุกอย่างที่คุณจะใช้บนเวทีหรือในสตูดิโออยู่ใน Performance ทั้งหมด ไม่ว่าจะเป็น Layer, Split, Scene หรือ Arpeggio

ทุกรุ่น

Performance คืออะไร?

Performance คือ "เพลง" หรือ "Patch" หนึ่งชุดที่รวม Part 1-8 (MODXM: 1-16) เข้าด้วยกัน แต่ละ Performance สามารถมีได้:

  • 🎹 เสียง Piano Layer กับ Strings
  • 🎸 Split Bass ซ้ายมือ / Organ ขวามือ
  • 🎤 เสียงเดียว (Single) เช่น Grand Piano เต็มๆ
  • 🎛️ Arpeggio เล่นตาม Chord อัตโนมัติ
  • 🔄 Motion Sequencer ควบคุม Filter/Volume อัตโนมัติ
  • 🎚️ 8 Scene (ตั้งค่าพร้อม Switch ได้ทันที)

โครงสร้าง Performance

Performance
├── Common Settings (ตั้งค่าส่วนกลาง)
│   ├── Tempo, Time Signature
│   ├── Arpeggio (สำหรับ Performance ทั้งหมด)
│   ├── Scene 1-8 (Snapshot การตั้งค่า)
│   ├── Super Knob (ควบคุมหลาย Parameter พร้อมกัน)
│   └── Master Effects, Master EQ
│
├── Part 1 → Sound (AWM2/FM-X/AN-X)
│   ├── Note Range (Low-High Limit)
│   ├── Velocity Range
│   ├── Volume, Pan, Detune
│   ├── Insertion Effect A, B
│   ├── Reverb Send, Variation Send
│   └── MIDI Channel, Receive Switch
│
├── Part 2-8 (เหมือน Part 1)
│
└── System Effects
    ├── Reverb (Hall/Room/Plate)
    └── Variation (Delay/Chorus/Flanger)

Home Screen — หน้าจอหลัก

เมื่อ Load Performance แล้ว หน้าจอจะแสดง:

  • ชื่อ Performance — ด้านบน
  • Part Slots — แท่งสีแสดง Part 1-8 ที่เปิดอยู่ พร้อม Volume Level
  • Scene Buttons — 1-8 ด้านล่าง
  • Arpeggio Indicator — แสดงว่า Arpeggio เปิดอยู่ไหม
  • Assign Knobs 1-8 — ควบคุม Parameter ที่กำหนดไว้

การทำ Single (เสียงเดียว)

Single คือ Performance ที่มีเพียง 1 Part เปิดอยู่ ครอบคลุมทั้ง Keyboard

  1. 1
    กด CATEGORY SEARCH เลือก Category เช่น Piano
  2. 2
    เลือก Performance ที่ต้องการ กด ENTER 2 ครั้ง
  3. 3
    ตรวจสอบ Home Screen — ถ้ามีเพียง Part 1 สว่างอยู่ = Single Mode

การทำ Layer (เสียงซ้อน)

Layer คือการเปิดหลาย Part พร้อมกัน ทั้งหมดครอบคลุม Keyboard เดียวกัน เช่น Piano + Strings

  1. 1
    Load Performance ที่ต้องการเป็น Layer พื้นฐาน เช่น "CFX Full Concert Grand"
  2. 2
    เพิ่ม Part 2 — แตะ Part 2 บน Home Screen → กด CATEGORY SEARCH เลือก "Slow Strings"
  3. 3
    ตั้ง Note Range Part 2 → Part Edit → Note Low = C-2, Note High = G8 (ครอบคลุมทั้ง Keyboard)
  4. 4
    ปรับ Volume สมดุล Part 1 (Piano) = 80-100, Part 2 (Strings) = 40-60 → ลอง Balance ตามชอบ
  5. 5
    กด STORE บันทึก Performance ใหม่ เลือก Slot ว่าง ตั้งชื่อ
🎵

Layer ที่นิยมสำหรับวงดนตรี

Piano + Slow Strings Pad → เสียงเต็มสำหรับ Ballad
Organ + Brass Short → โน้ตสั้นมีผลทันที เหมาะ Funk/R&B
Synth Lead + Octave Synth → Lead Sound สำหรับ Solo

การทำ Split (แบ่ง Keyboard)

Split คือการแบ่ง Keyboard ออกเป็นส่วนๆ แต่ละ Part ตอบสนองต่อส่วนต่างกัน

ตัวอย่าง: Bass ซ้าย / Piano ขวา

  1. 1
    Part 1 = Piano: Note Low = C4, Note High = G8 (ครึ่งขวา)
  2. 2
    Part 2 = Bass: Note Low = C-2, Note High = B3 (ครึ่งซ้าย)
  3. 3
    ตั้ง Octave Shift สำหรับ Bass: Part Edit → Pitch → Octave = -1 หรือ -2 เพื่อให้เล่นได้ต่ำขึ้น
  4. 4
    ตั้ง Mono Mode สำหรับ Bass: Part Edit → Mono → On เพื่อ Legato Bass

Scene — Snapshot การตั้งค่า

Scene คือการบันทึก "สถานะ" ของ Performance ณ ขณะนั้นไว้ใน Slot 1-8 สามารถ Switch ระหว่าง Scene ได้ทันทีขณะเล่น

สิ่งที่ Scene จำไว้:

  • Part Volume, Pan
  • Arpeggio On/Off, Arpeggio Type
  • Assign Knob Values
  • Super Knob Position
  • Motion Sequencer On/Off
  • Part Mute Status
🎭

การใช้ Scene สำหรับวงดนตรี

Scene 1 = Verse (Piano เต็ม, Strings เบา)
Scene 2 = Chorus (Piano+Strings เต็ม, Arpeggio เปิด)
Scene 3 = Bridge (Pad ลง Solo Synth ขึ้น)
กด Scene Button 1-8 ระหว่างเล่น เปลี่ยนได้ทันทีไม่มีหยุด

กด Scene อย่างไร?

  1. 1
    ปรับค่าที่ต้องการสำหรับ Scene 1 (Volume, Arpeggio, Knobs)
  2. 2
    กด SHIFT + Scene 1 เพื่อ Store Scene 1
  3. 3
    ปรับค่าใหม่สำหรับ Scene 2 → กด SHIFT + Scene 2
  4. 4
    ระหว่างเล่น กด Scene 1 หรือ Scene 2 เพื่อสลับทันที

Quick Edit — แก้ไขรวดเร็วจาก Home Screen

แตะ Part บน Home Screen ค้างไว้ จะแสดง Quick Edit Panel:

  • Volume Knob
  • Pan Knob
  • Cutoff (Filter)
  • Resonance
  • Reverb Send
  • Mute / Solo toggle

ใช้ Quick Edit ปรับเสียงเร็วโดยไม่ต้องเข้าเมนู Edit ลึก

Super Knob

Super Knob คือ Knob สีทองขนาดใหญ่ที่ควบคุม Parameter หลายตัวพร้อมกัน กำหนดได้ว่าจะควบคุมอะไร เช่น:

  • หมุน Super Knob → Part 1 Cutoff เพิ่ม + Part 2 Reverb Send ลด + Part 3 Volume เพิ่ม
  • เหมาะสำหรับ Dramatic Moment บนเวที
✅
