บทที่ 07 — Performance

Parts & Mixing

การตั้งค่า Parts ในรายละเอียด — Volume, Pan, EQ, Note Limit, Velocity, Mono/Poly, Detune และ MIDI Channel ที่ช่วยให้เสียงทุก Part อยู่ในตำแหน่งที่สมบูรณ์แบบ

ทุกรุ่น

เข้า Mixing Mode

Mixing Mode คือมุมมองที่แสดงทุก Part พร้อมกันใน Channel Strip เหมือน Mixer:

  1. 1
    จาก Home Screen กดปุ่ม PERFORMANCE แล้วเลือก Mixing หรือ แตะ Mixing ที่ Touchscreen
  2. 2
    หน้าจอแสดง Channel Strip ของ Part 1-8 เรียงเป็นแนว สามารถ Swipe ซ้าย/ขวาเพื่อดู Part เพิ่มเติม
  3. 3
    แตะ Part ไหนก็ได้เพื่อเปิด Part Setting Detail

Volume และ Pan

Part Volume

ปรับ Volume ของ Part นั้น (0-127) ใน Mixing:

  • ลาก Fader บน Touchscreen หรือใช้ DATA DIAL
  • ค่า Default = 100
  • ไม่ควรตั้ง Volume เต็ม 127 ทุก Part — เพราะรวมกันจะ Clip ให้ตั้งไว้ที่ 80-100 แล้วปรับ Balance

Part Pan

ปรับตำแหน่ง Stereo (L63 – Center – R63):

เสียงPan แนะนำเหตุผล
Piano (หลัก)Center (0)เป็นศูนย์กลาง ฟังทั่วถึง
String PadL10–R10 (กว้างเล็กน้อย)ให้ Stereo Width โดยไม่รบกวน Piano
OrganCenter หรือ L5Organ มักอยู่ตรงกลาง
Brass SectionL15–R15Brass มี Stereo Width ตามธรรมชาติ
BassCenter หรือ L5Bass ควรอยู่กลาง ไม่แตก Stereo มาก
Synth LeadCenterLead ต้องชัด อยู่กลาง

Note Limit — กำหนดช่วง Keyboard

Note Limit ตั้งว่า Part นี้ตอบสนองต่อโน้ตใด เป็นพื้นฐานของการทำ Split และ Layer ที่ซับซ้อน:

วิธีตั้งค่า

  1. 1
    เข้า Part Edit → Part Settings → Note Limit
  2. 2
    ตั้ง Note Low = โน้ตต่ำสุดที่ Part นี้รับ เช่น C-2
  3. 3
    ตั้ง Note High = โน้ตสูงสุดที่ Part นี้รับ เช่น B3
🎹

เทคนิค Split ด้วย Note Limit

Part 1 Piano: Note Low = C4, Note High = G8
Part 2 Bass: Note Low = C-2, Note High = B3
ผลคือ เล่นซ้ายได้ยินเสียง Bass เล่นขวาได้ยินเสียง Piano

Velocity Limit — ควบคุมด้วยแรงกด

Velocity Limit ตั้งว่า Part นี้ตอบสนองเมื่อกด Key ด้วย Velocity เท่าไร ช่วยสร้าง Velocity-based Layer:

การใช้งานVelocity LowVelocity Highผลที่ได้
Piano (กดเบา)180เสียง Piano นุ่มอย่างเดียว
Piano + Brass (กดแรง)80127กดแรงมาก Brass เข้ามา
Ensemble (กดกลาง-แรง)60127Strings เข้าถ้ากดปานกลางขึ้นไป

Mono / Poly Mode

Poly Mode (ค่า Default)

เล่นได้หลายโน้ตพร้อมกัน เหมาะกับ Piano, Chord, Pad, Strings ทุกอย่างที่ต้องการ Harmony

Mono Mode

เล่นได้เสียงเดียว โน้ตที่กดหลังสุดจะชนะ (Last Note Priority) เหมาะกับ:

  • Bass: Bass Line มักเล่น Mono เพื่อความชัดเจน
  • Synth Lead / Monophonic Lead: เสียง Lead ที่มีเค้า Analog Synth
  • Flute / Oboe / Solo Wind: เครื่องลมเล่นได้เสียงเดียว

วิธีตั้ง Mono: Part Edit → Part Settings → Mono/Poly → เลือก Mono

Portamento

Portamento ทำให้โน้ตไหลจากตัวหนึ่งไปอีกตัวหนึ่ง ใช้มากกับ Lead Synth, Violin, Vocal:

  • Porta Sw → On/Off
  • Porta Mode → Rate (ความเร็วคงที่) หรือ Time (เวลาคงที่)
  • Porta Time → 0 (เร็วมาก/ทันที) – 127 (ช้ามาก)
  • Porta Step → ไหลแบบ Glissando (เป็นขั้น) หรือ Continuous (ไหลต่อเนื่อง)
🎵

Portamento กับ Lead Guitar

ตั้ง Portamento Time ประมาณ 20-40 กับ Mono Lead Synth จะได้เสียงไหลเหมือน Guitar Bend หรือ Violin Glide ที่นักดนตรีชื่นชอบ

Detune

Detune ปรับ Pitch ของ Part ขึ้นหรือลงเป็น Cents (100 Cents = 1 Semitone):

  • ใช้ Part 1 Detune = 0, Part 2 Detune = +7 Cents สำหรับ Chorus Effect (Layer 2 Part เดียวกัน)
  • ใช้ Detune สร้าง "Wide Strings" — Layer 2 Part Strings, Detune ±5-10 Cents
  • Range: -100 ถึง +100 Cents

Part EQ

แต่ละ Part มี 3-Band Parametric EQ ในตัว ก่อนส่งออกไปยัง System Effects:

เสียงLowMidHighเหตุผล
Piano00+2เพิ่มความสดใส
Bass+30-3เน้น Low, ลด Treble ไม่จำเป็น
Strings Pad-30-2ตัด Low ที่ทับกับ Piano/Bass
Brass0+2 (2-3kHz)+1เพิ่ม Presence
Organ0+1+2เพิ่มความสว่าง

Mute และ Solo

Mute Part

ปิดเสียง Part ชั่วคราว (Part ยังคงอยู่ใน Performance แต่ไม่มีเสียงออก):

  • Home Screen → แตะ Part Slot ค้าง → เลือก Mute
  • หรือ Mixing → แตะ Part → ปุ่ม Mute (M)

Solo Part

ฟังเฉพาะ Part นั้นโดยปิด Part อื่นทั้งหมด:

  • Home Screen → แตะ Part Slot ค้าง → เลือก Solo
  • ใช้สำหรับ Troubleshoot เสียง หรือ ฟัง Part เดียวชัดๆ

Receive MIDI Channel

Each Part ตั้งได้ว่ารับ MIDI Channel ไหน (1-16 หรือ All):

  • Part Edit → MIDI → Receive Ch → เลือก Channel 1-16
  • ใช้เมื่อมี Controller ภายนอกส่ง MIDI แยก Channel ให้แต่ละ Part
  • ถ้าเล่นจาก Keyboard ของ MODX เอง ค่า Common Channel ควบคุมทุก Part
✅
