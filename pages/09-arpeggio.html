บทที่ 09 — Advanced

Arpeggio & Motion Sequencer

Arpeggio ของ MODX ไม่ใช่แค่ Up/Down มี Pattern กว่า 10,000 รูปแบบ ส่วน Motion Sequencer เปลี่ยน Effect/Volume/Filter อัตโนมัติตาม Sequence Loop สร้างความเคลื่อนไหวที่ไม่มีใครเทียบ

ทุกรุ่น

Arpeggio คืออะไร?

Arpeggio ใน MODX เปลี่ยน Chord ที่กดค้างไว้เป็น Pattern Sequence อัตโนมัติ ไม่ใช่แค่ Up/Down แต่มี Pattern ครบทุก Genre:

  • 🎵 Classic Arp (Up, Down, Up/Down, Random)
  • 🥁 Rhythm Pattern (กลอง Pattern ที่แปลงเป็น MIDI)
  • 🎸 Chord Strum Pattern (จำลองการ Strum Guitar)
  • 🎹 Sequencer-style Phrase

การเปิด Arpeggio

  1. 1
    กดปุ่ม ARP ON/OFF บน Front Panel (หรือแตะ Arp บน Home Screen)
  2. 2
    กด Chord บน Keyboard — Arpeggio Pattern เริ่มเล่น
  3. 3
    ปล่อย Chord เมื่อต้องการหยุด หรือ กด ARP HOLD เพื่อให้ Pattern วน Loop ต่อเนื่องแม้ปล่อย Key

การเลือก Arpeggio Pattern

  1. 1
    เข้า Performance Edit → Part Edit → แท็บ Arpeggio
  2. 2
    กด Category Search ใน Arpeggio Tab → เลือก Category
  3. 3
    Category แนะนำ: Piano → Arp, Bass → Groove, Seq → Analog, Perc → Drums

Arpeggio Parameters สำคัญ

Parameterค่า/ตัวเลือกคำอธิบาย
Arp On/OffOn / Offเปิด/ปิด Arpeggio สำหรับ Part นี้
Arp HoldOn / OffOn = Pattern วนแม้ปล่อย Key
Tempo1-300 BPMSpeed ของ Pattern (หรือ Sync กับ MIDI Clock)
Grid1/32, 1/16, 1/8, 1/4...ขนาดโน้ตในแต่ละ Step
Swing0-100%เพิ่ม Swing/Groove ให้ Pattern ไม่ตรงนาฬิกา
Velocity Rate0-200%ปรับ Velocity ของ Pattern (กดเบา/แรงเท่าไร)
Note Length0-200%ความยาว Note แต่ละตัว
Chord MatchOn / OffOn = Pattern ปรับโน้ตตาม Chord ที่กดอัตโนมัติ
Arpeggio TypeUp, Down, Up/Down, Random, Phrase...รูปแบบ Pattern พื้นฐาน

Arpeggio ต่อหลาย Part พร้อมกัน

สามารถตั้ง Arpeggio แยกต่อ Part ได้ (Part 1-8 แต่ละตัวมี Arpeggio Pattern ของตัวเอง) ทำให้สร้าง Complex Multi-Part Pattern:

🎵

ตัวอย่าง: Dance Pattern 4 Part

Part 1 (Bass) → Bass Pattern 1/8 Note Groov
Part 2 (Piano) → Chord Strum 1/16 Note
Part 3 (Strings) → Slow Pad (ไม่มี Arp)
Part 4 (Drums) → 4/4 Kick+Snare Pattern
กด Chord เดียว ทุก Part ออกมาพร้อมกันเป็น Full Track!

Motion Sequencer — ความเคลื่อนไหวอัตโนมัติ

Motion Sequencer (MS) คือ LFO ชั้นสูงที่ควบคุม Parameter ใดก็ได้ใน MODX ด้วย Sequence Loop ที่กำหนดค่าเองได้ทุก Step

โครงสร้าง Motion Sequencer

Part มี Motion Sequencer 4 Lanes
Lane 1 → ควบคุม: Part Volume (ตัวอย่าง)
Lane 2 → ควบคุม: Filter Cutoff
Lane 3 → ควบคุม: Reverb Send
Lane 4 → ควบคุม: LFO Speed

แต่ละ Lane มี 16 Steps กำหนดค่าแต่ละ Step (0-127)
Loop ต่อเนื่อง Sync กับ Tempo

วิธีตั้งค่า Motion Sequencer

  1. 1
    เข้า Performance Edit → Part Edit → Motion
  2. 2
    เปิด Lane 1 → กำหนด Target เช่น Filter Cutoff
  3. 3
    ตั้ง Step 1-16 — แตะแถบ Step บน Touchscreen เพื่อกำหนดค่า 0-127
  4. 4
    ตั้ง Speed เช่น 1/4 Note หรือ 1/8 Note Sync กับ Tempo
  5. 5
    เปิด Motion Sequencer On บน Front Panel หรือ Home Screen

Motion Sequencer Targets ที่น่าสนใจ

🎚️
Filter Cutoff
เปิด-ปิด Filter ตาม Step ได้เสียง Wah/Filter Sweep อัตโนมัติ เหมาะกับ Synth, Pad
🔊
Part Volume
Volume ขึ้น-ลงตาม Step สร้าง Tremolo Effect หรือ Swell เอง
↔️
Part Pan
Auto Pan ซ้าย-ขวา ตาม Step ได้เสียงวนรอบ Stereo
🎛️
Effect Parameters
ปรับ Reverb Send, Delay Time, Chorus Depth ตาม Step อัตโนมัติ

Super Knob และ Assign Knobs

Assign Knobs 1-8

Knob 8 ตัวบน Front Panel กำหนดได้ว่าแต่ละตัวควบคุมอะไร:

  1. 1
    กด SHIFT + Assign Knob → เปิด Assign Edit
  2. 2
    เลือก Destination Part และ Parameter เช่น Part 1 → Filter Cutoff
  3. 3
    กำหนด Range Min/Max ของ Knob

Super Knob

Knob สีทองขนาดใหญ่ — กำหนดให้ควบคุม Assign Knob 1-8 พร้อมกันได้ หรือกำหนด Destination แยกต่างหาก:

⭐

ตัวอย่าง Super Knob สำหรับวง

Super Knob หมุนไปทาง Max:
→ Part 1 Piano Volume เพิ่ม
→ Part 2 Strings Volume เพิ่ม
→ Part 3 Brass Volume เพิ่ม
→ System Reverb เพิ่ม
กด Super Knob ขณะ Climax เพลง — เสียงทุกอย่างขึ้นพร้อมกันทันที!

✅
