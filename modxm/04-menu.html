บทที่ 04 / 12

แนะนำเมนูทุกเมนู

บทนี้พาเดินทุกเมนูของ MODX ตามหน้าจอจริง อธิบายว่ากดอะไร → จอขึ้นอะไร → จิ้มตรงไหน → ไปหน้าใหม่แบบไหน จนครบทุก Branch

ระบบ Navigation ของ MODX

โครงสร้างเมนูหลัก

MODX ไม่มีเมนูย่อยแบบซ้อนกันหลายชั้นเหมือน Keyboard เก่า แต่ใช้ระบบ Tab + จิ้มหน้าจอ

ทุกหน้าจะมี Tab ด้านบน: จิ้ม Tab เพื่อสลับ Section / ใช้ Data Dial เลื่อนค่า / กด [BACK] หรือ [HOME] เพื่อออก

HOME Screen — หน้าแรกที่เห็นตอนเปิดเครื่อง

กด [HOME]

จอแสดงผล:

บนสุด: ชื่อ Performance ปัจจุบัน (เช่น "CFX + Strings") / กลางจอ: 8 Part Blocks แสดงชื่อเสียงแต่ละ Part + Level Meter / ล่าง: Super Knob Position + Scene 1-8 Indicator
จิ้มที่ Part Block ใดก็ได้ → Part นั้นถูก Select ไฟปุ่มสว่าง → ข้อมูล Part แสดงที่มุมล่างจอ
จิ้ม Performance Name (ข้อความบนสุด) → เปิดหน้า Category Search เพื่อเลือก Performance ใหม่

Category Search — เลือกเสียง Performance/Voice

กด [CATEGORY SEARCH]
จอเปลี่ยนเป็น Grid ของ Category: Piano / Organ / Strings / Brass / Synth Lead / Synth Pad / FM / Drum ฯลฯ
จิ้ม Category เช่น "Piano" → จอแสดง Sub-Category: Grand Piano, Upright, Electric Piano, Layer Piano
จิ้ม Sub-Category → จอแสดง List ชื่อ Performance ทั้งหมดใน Category นั้น

หมุน Data Dial หรือ Swipe รายการ → เลือก Performance ที่ต้องการ → จเสียงให้ฟัง Preview ขณะเดินผ่านรายการ

จิ้ม [OK] หรือกด Data Dial → โหลด Performance นั้น → จอกลับ HOME แสดงชื่อ Performance ใหม่

Performance Edit — แก้ไขเสียง

กด [EDIT]
จอแสดงหน้า Performance Edit มี 2 ระดับ: บนสุดคือ Tab "Common/Audio" และ Tab "Part 1" ถึง "Part 8" (แถบสีแตกต่างกันตามประเภท AWM2/FM-X)
จิ้ม Tab "Common/Audio" → เข้าตั้งค่าส่วนกลางของ Performance (Name, Effects chain, Arpeggio Master)
จิ้ม Tab "Part 1" (หรือ Part ที่ต้องการ) → เข้าหน้า Part Edit ของ Part นั้น

หน้า Part Edit (AWM2) — Tab ย่อยด้านบน

เมื่อเข้า Part Edit แล้ว จะเห็น Tab แถวบน
Tab: Element / OSC Tune / Filter / Amp / Mod/Control / Effect / Arpeggio
จิ้ม Tab "OSC Tune" → จอแสดง: Waveform ปัจจุบัน, Octave, Coarse Tune, Fine Tune, Note Range Low/High
จิ้ม Tab "Filter" → จอแสดง Slider: HPF Cutoff, LPF Cutoff, Resonance, Filter EG Depth, Filter EG ทั้ง 4 จุด (A/D/S/R) เป็น Graphical Envelope
จิ้ม Tab "Amp" → จอแสดง Graphical ADSR Envelope + Level + Pan ปรับโดยจิ้มที่จุด Handle ใน Envelope Graph แล้วลาก
จิ้ม Tab "Mod/Control" → จอแสดง Assign Matrix: Source (Mod Wheel, Pedal, Velocity ฯลฯ) → Destination (Cutoff, Pitch, Level ฯลฯ)

หน้า Part Edit (FM-X) — Tab ย่อยด้านบน

Tab ของ FM-X Part Edit
Tab: Algorithm / Operator (Op1-Op8) / LFO / Tune / Mod/Control / Effect / Arpeggio
จิ้ม Tab "Algorithm" → จอแสดงไดอะแกรม Algorithm ที่กำลังใช้ (รูป Flowchart ของ Operator) ด้านล่างมี < > เลือก Algorithm 1-88
จิ้ม Operator Block ใน Diagram → Highlight Operator นั้น → ด้านล่างแสดง Level, Frequency Ratio, Feedback ของ Op นั้น
จิ้ม Tab "Operator" → จอแสดงรายละเอียด Op ที่เลือก: Level, Freq Ratio, Detune, Envelope (Graph แบบ Line ลากได้)

Utility — ตั้งค่าระบบ

กด [UTILITY]
จอเปลี่ยนเป็นหน้า Utility มี 5 Tab แถวบน: General / Sound / MIDI / Contents / Update
จิ้ม Tab "General" → เห็น: Master Tune, Master Transpose, Keyboard Velocity, Screen Brightness, Auto Power Off
จิ้มค่า Master Tune → ค่าไฮไลต์สีน้ำเงิน → หมุน Data Dial เพื่อเปลี่ยน (หรือจิ้ม [+] [-] ใน Pop-up)
จิ้ม Tab "MIDI" → เห็น: MIDI IN/OUT Mode (USB/DIN), Local Control, Clock Source, Receive/Transmit Channel
จิ้ม Tab "Contents" → เห็นปุ่ม Load / Save / Delete สำหรับจัดการไฟล์ใน USB Drive

Live Set — โหมดรวม Performance สำหรับสด

กด [LIVE SET]
จอแสดง Live Set Grid: Performance Card 8 ช่องต่อ Row แสดงชื่อ Performance + สีที่กำหนด
จิ้ม Performance Card ใด → โหลด Performance นั้นทันที ไม่มี Lag เหมาะสำหรับสลับเพลงระหว่างแสดงสด
จิ้ม [EDIT] ขณะอยู่ใน Live Set → เข้าโหมดแก้ไข Live Set: ลากสลับตำแหน่ง, เปลี่ยนสี, ตั้งชื่อ Card

ปุ่ม [BACK] — กลับหน้าก่อนหน้า

กด [BACK] หรือจิ้ม ← ที่มุมบนซ้ายจอ

กลับไปหน้าก่อนหน้า 1 Step โดยไม่ปิด Performance ที่กำลังแก้ไข ถ้ากด [HOME] จะออกจากเมนูทั้งหมดกลับหน้าหลัก

ถ้าทำการแก้ไข Parameter แล้วกด [HOME] โดยไม่ [STORE] ก่อน — การแก้ไขจะหายไป! กด [STORE] ก่อนออกทุกครั้ง
💡 Navigation Shortcut สรุป

[HOME] = กลับหน้าหลักเสมอ | [EDIT] = เข้าแก้ไข Performance | [BACK] = ย้อน 1 ขั้น | ปุ่ม Part 1-8 = เลือก Part ตรงๆ | จิ้ม Tab ด้านบน = สลับ Section | หมุน Dial = เปลี่ยนค่า | กด Dial = Confirm
