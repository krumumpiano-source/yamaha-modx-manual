แผงปุ่มและอินเทอร์เฟส

MODX มีจอ TFT Color Touch Screen 7 นิ้ว ใช้ได้ทั้งการ จิ้มหน้าจอด้วยนิ้ว และ กด/หมุน Data Dial บทนี้อธิบายปุ่มทุกปุ่มและบอกว่าจอจะแสดงอะไรหลังกดแต่ละครั้ง

จอ Touch Screen — วิธีใช้งาน

วิธีที่ 1 — จิ้มหน้าจอ

ใช้ปลายนิ้วแตะที่ไอคอน, ปุ่ม, หรือค่าในจอโดยตรง เหมือน Smartphone ทั่วไป

ลาก (Swipe) ซ้าย-ขวาในบาง Menu เพื่อเลื่อนดู List ได้เช่นกัน

วิธีที่ 2 — Data Dial

หมุน Data Dial (ล้อกลมด้านขวาของจอ) เพื่อเลื่อนค่าขึ้น-ลง / เลือก Item ใน List

กด Data Dial ลงเพื่อ Confirm (เหมือนกด Enter)

วิธีที่ 3 — Inc/Dec Buttons

ปุ่ม [INC] / [DEC] (บางรุ่นมี) สำหรับเพิ่ม-ลดค่าทีละ 1 เหมาะเมื่อต้องการความแม่นยำ

ปุ่มหลักด้านซ้ายบนของจอ

ปุ่มกดแล้วจอขึ้นอะไรใช้ทำอะไร
[HOME] หน้า HOME ▶ แสดงชื่อ Performance, Parts List, Super Knob, 8 Scene Buttons กลับหน้าหลักได้เสมอ กด ตอนหลงในเมนูลึก
[PERFORMANCE] หน้า Performance Play ▶ แสดงรูป Piano Roll + Part List + Level Meter กลับมาโหมดเล่นปกติ
[EDIT] หน้า Performance Edit ▶ แสดง Tab: Common/Audio, Part 1-8 เข้าแก้ไขเสียง
[CATEGORY SEARCH] หน้า Category Search ▶ แสดงรายการ Category เป็น Grid: Piano, Organ, Strings ... ค้นหาและเลือกเสียง
[UTILITY] หน้า Utility ▶ แสดง 5 Tab: General, Sound, MIDI, Contents, Update ตั้งค่าระบบทั้งหมด
[STORE] Pop-up "Store" ▶ แสดงช่องเลือก Bank, Number, Name + ปุ่ม OK/Cancel บันทึก Performance

Super Knob — ปุ่มใหญ่กลางเครื่อง

หมุน Super Knob ขณะเล่น

จอแสดงผล: วงกลมใหญ่ตรงกลางจอหมุนตาม → ค่า Super Knob เปลี่ยน 0–127 มุมบนซ้ายจอแสดงตัวเลขค่า Knob ปัจจุบัน

Super Knob = 64 / Parameter ที่ Assign ไว้เปลี่ยนค่าพร้อมกัน
กด Super Knob ค้าง

จอแสดง Pop-up: "Assign Super Knob" → เห็นรายการ Parameter ที่ Knob นี้ควบคุมอยู่ทั้งหมด

จิ้ม Parameter ในรายการเพื่อดูว่า Map กับอะไร หรือจิ้ม [EDIT] เพื่อแก้ Assign

8 Assignable Knobs (Knob 1–8)

หมุน Knob 1–8

ค่าที่ควบคุมขึ้นกับว่า Assign ไว้กับ Parameter อะไร ค่า Default ของ Knob 1-8 คือ: Volume, Cutoff, Resonance, LFO Rate, LFO Depth, Reverb, Chorus, Filter Env Depth

มุมล่างจอแสดงแถบ Parameter Name + ค่าปัจจุบันเป็น Pop-up ชั่วคราว 2 วินาที
กด [ASSIGN] เพื่อเปลี่ยนสิ่งที่ Knob ควบคุม

กด [SHIFT] + [ASSIGN] → จอเปิดหน้า Assignable Knob Config

จิ้ม Knob Number (1-8) ที่ต้องการ → จิ้ม Destination เพื่อเลือก Parameter ที่จะ Control

8 Scene Buttons (สีต่างๆ)

กด Scene Button 1–8

Scene Button ที่เลือกอยู่จะ สว่างขึ้น สีเต็ม / Scene อื่นจะหรี่ลง

จอ HOME แสดงไฮไลต์ที่ Scene ที่กำลังใช้งาน + ค่า Super Knob และ Part Volume อาจเปลี่ยนตาม Scene ที่บันทึกไว้
กด Scene ค้าง 1 วินาที

บันทึกสถานะปัจจุบัน (Volume, Super Knob position) ลง Scene นั้น

Pop-up: "Scene [number] Stored" แสดงขึ้น 1 วินาที แล้วหายไปเอง

Part Select Buttons (Part 1–8)

กด Part 1–8

Part ที่กดจะ ไฟสว่าง ขึ้น

จอ HOME แสดง Part นั้นถูก Highlight / ถ้าอยู่ในหน้า Edit จะ Jump ไปที่ Part Edit ของ Part นั้นทันที
กด [SHIFT] + Part Button = Mute/Unmute

ถ้า Part นั้น Active → กด Shift+Part = Mute (ไฟ Part กะพริบสีแดง) / กดอีกครั้ง = Unmute

ไฟปุ่ม Part กะพริบแดง = Mute / สว่างปกติ = Active
จิ้มที่ Part Icon ในหน้าจอ HOME ก็ทำได้เช่นกัน — จิ้มครั้งเดียว = Select, สองครั้ง = Mute Toggle

Transport Buttons

ปุ่มกดแล้วเกิดอะไร
[▶ PLAY] Arpeggio / Motion Sequencer เริ่มเล่น — จอแสดง Beat Counter กะพริบ + ตำแหน่ง Bar:Beat
[■ STOP] Arpeggio/Motion Seq หยุด — จอ Beat Counter หยุดนิ่ง
[● REC] เข้าโหมด Record Motion Sequence — จอเปลี่ยนสีพื้นหลังเป็นแดง + ข้อความ "Recording..." กระพริบ
[TAP TEMPO] กด 4 ครั้งตาม Beat → จอแสดง BPM ที่คำนวณได้ทันที: "Tempo = 120 BPM"

ด้านหลังเครื่อง — Output & Ports

OUTPUT L/MONO + R

6.3mm TRS ต่อ Mixer หรือ Active Speaker ถ้าต่อแค่ L/MONO จะได้เสียง Mono Mix ของทั้ง 2 Channel

MIDI IN / MIDI OUT (5-pin DIN)

สำหรับต่ออุปกรณ์ MIDI รุ่นเก่าที่ไม่มี USB เช่น Drum Machine, MIDI Controller รุ่นเก่า

USB-B (to Host)

ต่อ Computer ผ่าน USB Cable — MODX ส่ง MIDI เท่านั้น (ไม่มี USB Audio) ต้องติดตั้ง Driver ก่อนใช้งานครั้งแรก

Sustain / Assignable Pedal Jack

ต่อ FC3A (Sustain) หรือ FC7 (Expression Pedal) ตั้งค่าที่ Utility > General > Foot Pedal

USB-A (to Device)

ด้านหน้าเครื่อง ต่อ USB Flash Drive FAT32 สำหรับ Save/Load Performance และ Update Firmware

💡 เคล็ดลับ: Short Cut บนจอ

จิ้มค้างที่ Parameter Name ใดก็ตามในจอ 1 วินาที → Pop-up "Assign to Knob?" จะปรากฏ → จิ้ม Yes แล้วหมุน Knob ที่ต้องการ → Parameter นั้น Assign ให้ Knob ได้ทันที ไม่ต้องเข้าเมนู!
