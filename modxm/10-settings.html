Utility & การตั้งค่าทั้งหมด

Utility ของ MODX เก็บการตั้งค่าทั้งหมดของเครื่อง ตั้งแต่ Master Tune ไปจนถึง Firmware Update

วิธีเข้า Utility

ขั้นที่ 1

กด [UTILITY] บนตัวเครื่อง → หน้า Utility จะปรากฏ

การตั้งค่าสำคัญ

General Settings

Settingค่าแนะนำคำอธิบาย
Master Tune440.0 Hzปรับ Tune รวมทั้งเครื่อง ± 50 cents
Master Transpose0Transpose ทั้งเครื่อง (semitone)
Keyboard VelocityNormalความไวของ Key Velocity

MIDI Settings

Settingค่าแนะนำคำอธิบาย
MIDI IN/OUTUSBเลือก USB สำหรับต่อ Computer; DIN สำหรับอุปกรณ์เก่า
Local ControlOnOff ถ้าส่ง MIDI ผ่าน DAW แล้ว Route กลับ (หลีกเลี่ยง Double Trigger)
Clock SourceInternalเปลี่ยนเป็น MIDI ถ้า Sync กับ DAW Tempo

Factory Reset

วิธี Reset

Utility → Initialize → ตัวเลือก: All (ทุกอย่าง) / User Voice (User Bank เท่านั้น) / System (แค่ System Settings) — Backup ก่อนเสมอ!

Firmware Update

ขั้นที่ 1

ดาวน์โหลด Firmware ล่าสุดจาก yamaha.com → ใส่ใน USB Root Directory (ไม่อยู่ใน Folder)

ขั้นที่ 2

เสียบ USB เข้าเครื่อง → Utility → Update Firmware → กด Accept → รอ Restart อัตโนมัติ

⚠️ สำคัญ

อย่าปิดเครื่องระหว่าง Firmware Update — ถ้าไฟดับ/ปิดเครื่อง เครื่องอาจ Brick ได้ ต้องส่งซ่อม
