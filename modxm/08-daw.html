ต่อ DAW & MIDI Setup

MODX+ มี USB Audio Interface 2in/2out ในตัว สามารถส่ง Audio + MIDI ผ่าน USB เดียวไปยัง DAW ได้เลย ไม่ต้องซื้อ Audio Interface แยก

วิธีต่อ MODX+ กับ DAW (USB Audio + MIDI รวมกัน)

ขั้นที่ 1

ต่อ USB-B Cable จาก MODX+ → Computer — USB เส้นเดียวส่งทั้ง Audio และ MIDI

ขั้นที่ 2 (Windows)

ติดตั้ง Yamaha Steinberg USB Driver จาก yamaha.com → Restart Computer หลังติดตั้ง

ขั้นที่ 2 (Mac)

Mac รู้จัก MODX+ เป็น Core Audio Device อัตโนมัติ ไม่ต้องติดตั้ง Driver เพิ่มเติม

ขั้นที่ 3

กด [UTILITY] → Settings → USB I/O → ตั้ง USB Mode = MIDI + Audio (ต่างจาก MODX ที่มีแค่ MIDI)

ขั้นที่ 4

เปิด DAW → Audio Settings → เลือก Driver = Yamaha MODX+ ASIO (Windows) หรือ Yamaha MODX+ (Mac Core Audio)

ขั้นที่ 5

ใน DAW เพิ่ม Audio Track → Input = MODX+ In 1-2 → Arm → เล่น = Audio ส่งเข้า DAW ทันที

ตัวอย่าง Setup ใน Cubase / Ableton

Cubase

ขั้น

Studio → Studio Setup → VST Audio System → เลือก ASIO Driver = Yamaha Steinberg USB Driver → OK → Add Audio Track → Input Bus = Stereo In (MODX+)

Ableton Live

ขั้น

Preferences → Audio → Driver Type = ASIO (Windows) / Core Audio (Mac) → Audio Input Device = Yamaha MODX+ → เปิด Input 1/2 → OK

Monitoring Audio ผ่าน MODX+

USB Audio In

DAW → MODX+ USB Audio In → เสียง Monitor ออกจาก Speaker/Headphone ของ MODX+ ได้ เหมาะสำหรับ Monitor DAW โดยไม่ต้องซื้อ Monitor Speaker แยก (ใช้ Headphone ที่ MODX+)

💡 Tips: Sample Rate

ตั้ง Sample Rate ให้ตรงกันทั้ง DAW และ MODX+: แนะนำ 48kHz สำหรับ Recording ทั่วไป หรือ 96kHz ถ้าต้องการ Hi-Res
