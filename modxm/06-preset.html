การจัดการ Preset & Library

บทนี้อธิบาย ทีละขั้นตอน วิธีบันทึกเสียง (Store), ตั้งชื่อ, สำรองข้อมูลไว้ USB, และโหลด Library กลับมา — พร้อมบอกว่าจอแสดงอะไรแต่ละขั้น

การบันทึก Performance (Store) — ขั้นตอน 100%

ขั้นที่ 1 — แก้ไขเสียงจนพอใจแล้ว

แก้ไข Performance ใดก็ตามในหน้า Edit เสร็จแล้ว → สังเกตชื่อ Performance บนสุดจอจะมีเครื่องหมาย "*" (ดอกจัน) ต่อท้าย = แสดงว่ามีการเปลี่ยนแปลงที่ยังไม่ได้บันทึก

บนสุดจอ: "CFX + Strings*" — ดาวหมายถึง Unsaved Changes
ขั้นที่ 2 — กด [STORE]

กด [STORE] บนตัวเครื่อง

Pop-up "Store Performance" เปิดขึ้นกลางจอ มีส่วนต่างๆ:
• "Bank" — เลือก User Bank (User 1 ถึง User 8)
• "Number" — เลือก Slot 001-128
• "Name" — ชื่อ Performance ปัจจุบัน
• ปุ่ม [Rename] / [OK] / [Cancel]
ขั้นที่ 3 — เลือก Bank และ Slot
จิ้มที่ "Bank" → Pop-up เล็กๆ เลือก User1 ถึง User8 → จิ้ม Bank ที่ต้องการ
จิ้มที่ "Number" → หมุน Data Dial เลื่อนดู Slot / มองซ้ายจะเห็นชื่อ Performance ที่อยู่ใน Slot นั้นอยู่แล้ว (ถ้าว่างจะขึ้น "Init Performance")
ถ้าเลือก Slot ที่มี Performance อยู่แล้ว → Store จะ Overwrite ทับทันที ไม่มี Undo — ระวัง!
ขั้นที่ 4 — ตั้งชื่อ (Rename)
จิ้มปุ่ม [Rename] → หน้า Keyboard On-Screen เปิดขึ้น
จอแสดง Virtual Keyboard ภาษาอังกฤษ (ตัวพิมพ์ใหญ่/เล็ก/ตัวเลข/สัญลักษณ์ สลับได้ด้วยปุ่ม Shift/Symbol) + ช่องชื่อด้านบนที่มี Cursor กะพริบ
จิ้มตัวอักษรบน Virtual Keyboard ทีละตัว / จิ้ม [←] เพื่อลบ / จิ้ม [SPACE] เพื่อเว้นวรรค
จิ้มปุ่ม [OK] เมื่อพิมพ์ชื่อเสร็จ → กลับมาที่หน้า Store Pop-up ชื่อใหม่ปรากฏในช่อง Name
ขั้นที่ 5 — ยืนยันการบันทึก
จิ้มปุ่ม [OK] ในหน้า Store Pop-up
แถบ Progress เล็กๆ ปรากฏขึ้นครู่หนึ่ง → จอกลับหน้า HOME → ชื่อ Performance ด้านบนไม่มี "*" แล้ว = บันทึกสำเร็จ

Backup ทุกอย่างไปยัง USB Flash Drive

ขั้นที่ 1 — เสียบ USB Flash Drive

เสียบ USB Flash Drive (Format FAT32) เข้าพอร์ต USB-A ด้านหน้าตัวเครื่อง (ใต้จอด้านซ้าย)

จอมุมบนขวาแสดงไอคอน USB สีขาวปรากฏขึ้น = เครื่องรู้จัก USB แล้ว (รอ 3-5 วินาที)
ขั้นที่ 2 — เข้า Utility > Contents > Save

กด [UTILITY]

จอแสดงหน้า Utility Tab: General / Sound / MIDI / Contents / Update
จิ้ม Tab "Contents"
จอแสดง: ปุ่ม Load / Save / Delete + ด้านล่างแสดงรายการไฟล์ใน USB (ถ้ามีอยู่แล้ว) / ด้านขวาแสดงขนาดหน่วยความจำ USB
จิ้มปุ่ม [Save]
ขั้นที่ 3 — เลือกสิ่งที่จะ Save
Pop-up "Save" ปรากฏ มี 3 Option:
• All — บันทึกทุกอย่าง: User Performances + System Settings
• Library — บันทึกเฉพาะ Performance Bank ที่เลือก
• Setting — บันทึกเฉพาะ System Settings
จิ้ม "All" เพื่อ Backup แบบครบถ้วน → ช่องตั้งชื่อไฟล์ปรากฏ → จิ้ม [Rename] ตั้งชื่อ (เช่น "Backup_2026") → จิ้ม [OK]
Progress Bar แสดงการ Save → เมื่อเสร็จ Pop-up "Completed" ปรากฏ → จิ้ม [OK] ปิด

โหลด Library จาก USB — ขั้นตอน

ขั้นที่ 1-2

เสียบ USB → กด [UTILITY] → จิ้ม Tab "Contents"

ขั้นที่ 3 — จิ้มปุ่ม [Load]
จอแสดง File Browser: รายการไฟล์ทั้งหมดใน USB (โฟลเดอร์และไฟล์ .X8L / .X8A / .Y2A ฯลฯ)
จิ้มโฟลเดอร์เพื่อเปิด → จิ้มไฟล์ .X8L ที่ต้องการ → จิ้มปุ่ม [Load]
ขั้นที่ 4 — เลือก Destination Bank
Pop-up "Load to Bank" ปรากฏ → เลือกว่าจะโหลดไปที่ User Bank ไหน (User 1-8)
จิ้ม Bank เป้าหมาย → จิ้ม [OK]
Progress Bar โหลด → เสร็จแล้ว Pop-up "Completed" → จิ้ม [OK] → Performance จาก Library พร้อมใช้งาน
การ Load จะ Overwrite ทับ Performance ทั้งหมดใน Bank นั้น ควร Save Backup ก่อนทุกครั้ง
💡 ดาวน์โหลด Preset ฟรีจาก Soundmondo

ไปที่ soundmondo.com → ค้นหา "MODX" → Download ไฟล์ .X8L → ใส่ใน USB → Load เข้าเครื่อง — มี Library จากนักดนตรีทั่วโลก ฟรี!
