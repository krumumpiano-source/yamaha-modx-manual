ภาพรวม MODX M คืออะไร

MODX M คือ Music Synthesizer รุ่นใหม่ล่าสุดของ Yamaha MODX Series มีคีย์บอร์ดในตัว (M6=61, M7=76, M8=88 คีย์) รวม 3 Sound Engine: AWM2 + FM-X + AN-X Virtual Analog ในเครื่องเดียว พร้อม Display Knobs 6 ปุ่ม, 8 Faders, ปุ่ม NAVIGATION ใหม่ และ USB Audio 2in/2out

รุ่นและคีย์บอร์ด — M6 / M7 / M8

แบ่ง 3 รุ่นตามขนาดคีย์บอร์ด
รุ่นจำนวนคีย์ประเภทคีย์
MODX M661 คีย์FSB (Balanced Hammer) Semi-weighted
MODX M776 คีย์FSB Semi-weighted
MODX M888 คีย์BHC (Balanced Hammer Compact) Graded Hammer

1. ระบบการนำทางและหน้าจอสั่งการ

ปุ่ม [NAVIGATION] — ใหม่ใน MODX M

กด [NAVIGATION] เพื่อดูภาพรวม Signal Flow ทั้งหมดของ Performance ปัจจุบัน

จอแสดง Block Diagram ของ Sound Engine — เห็นได้ชัดว่าเสียงไหลจาก Oscillator → Filter → Effect อย่างไร และจุดไหนมี Modulation เชื่อมอยู่ (ไฮไลต์สี)
จิ้มที่ Block ใดก็ได้บนจอ เช่น "Filter" หรือ "LFO" → กระโดดไปหน้าแก้ไขของ Block นั้นทันที ไม่ต้องวนผ่านเมนู
Display Knobs 6 ปุ่ม — อยู่ใต้จอ

MODX M มี Display Knobs 6 ปุ่ม เรียงใต้หน้าจอ แต่ละปุ่มจะเปลี่ยนหน้าที่ตามป้ายที่แสดงบนจอด้านบนของมัน

เมื่ออยู่หน้า Part Edit → Display Knob 1-6 แสดงป้าย: Level / Pan / Cutoff / Resonance / Attack / Release — หมุนได้ทันทีโดยไม่ต้องจิ้มหน้าจอก่อน

ข้อได้เปรียบ: ปรับค่าได้เร็วกว่าการจิ้มจอมาก และสัมผัสได้จริง ไม่ต้องก้มมองขณะเล่น

ปุ่ม [PAGE JUMP] — Quick Edit สู่ Deep Edit

ขณะอยู่ใน Quick Edit (การปรับค่าด่วนจากหน้า Performance) กด [PAGE JUMP]

จอกระโดดไปที่หน้า Edit เชิงลึก (Full Parameter Page) ของ Parameter นั้นทันที — เช่น กำลังปรับ Cutoff ในหน้า Quick → [PAGE JUMP] → เปิดหน้า Filter Edit แบบเต็มรูปแบบ
หลังแก้ไขเสร็จ จิ้มปุ่ม [EXIT] หรือกด [PERFORMANCE] เพื่อกลับหน้าเล่น

2. Sound Engines — 3 Engine ที่ต้องเข้าใจ

AWM2 — Sample-based สูงสุด 128 Elements ต่อ Part

AWM2 ใช้ Sample เสียงเครื่องดนตรีจริง (เปียโน, สตริง, ลม) ใน MODX M อัปจาก 8 Elements เป็น 128 Elements ต่อ 1 Part

กด [EDIT] → เลือก Part → จิ้ม Tab "AWM2" → เห็น Element List สูงสุด 128 รายการ แต่ละ Element มี Waveform, Filter, EG, LFO ของตัวเอง

ผลลัพธ์: เสียง Piano หรือ Strings ที่ Layer หลาย Velocity Layer + Round Robin ได้ละเอียดกว่าเดิมมาก ฟังดูสมจริงมากขึ้น

FM-X — 8 Operators, 88 Algorithms

FM-X สร้างเสียงด้วยการ Modulate ความถี่ระหว่าง Operator เหมาะกับเสียง Electric Piano, Bell, Bass แนว DX7 ไปจนถึงเสียงแปลกใหม่

กด [EDIT] → เลือก Part FM-X → จิ้ม Tab "Operator" → จอแสดง Block ของ Operator 1-8 พร้อม Algorithm Diagram ให้จิ้มเลือก Carrier/Modulator
จิ้ม Algorithm Number ที่มุมบนขวาของจอ → Pop-up แสดง 88 Algorithm ให้เลือก → จิ้ม Algorithm ที่ต้องการ → Block Diagram เปลี่ยนทันที
AN-X — Virtual Analog เฉพาะ MODX M

AN-X จำลองวงจร Analog Synth แบบ Digital สมบูรณ์ ประกอบด้วย:

  • OSC 1 + OSC 2 — ปรับ Waveform, Pulse Width, Wave Shaper, OSC Self Sync
  • OSC 3 — Modulate OSC 1/2 ด้วย FM หรือ Ring Modulation
  • Noise Generator — สร้างเสียง Breath, Wind, Hi-hat ได้
  • Filter 1 + Filter 2 — Route Output ของแต่ละ OSC เข้า Filter ได้อิสระ
กด [EDIT] → เลือก Part AN-X → จิ้ม Tab "OSC/Tune" → จอแสดง Block Diagram: OSC1, OSC2, OSC3, Noise → Filter1 → Filter2 → Amp พร้อม Knob ปรับแต่ละจุด

ได้เสียง: Acid Bass (OSC3 FM into OSC1 + Filter Env), Classic Lead (OSC Sync), Warm Pad (Detuned OSC1+2)

3. การควบคุมแบบ Real-time

Super Knob — ควบคุมร้อย Parameter ด้วยนิ้วเดียว

Super Knob ตรงกลางเครื่อง เมื่อ Assign แล้วสามารถควบคุม Parameter หลายสิบค่าพร้อมกันจากการหมุนครั้งเดียว

หมุน Super Knob → วงกลมใหญ่กลางจอหมุนตาม + Parameter ทุกตัวที่ Assign ไว้เปลี่ยนค่าตามสัดส่วนที่กำหนด (Depth)
กด [EDIT] → Tab "Common" → "Control" → "Super Knob" → จิ้ม [+] เพิ่ม Assign → เลือก Destination จาก Browser → ตั้ง Curve และ Depth

เคล็ดลับ: ใช้ FC7 Expression Pedal ต่อพอร์ต Assignable → Assign ให้คุม Super Knob → เล่นมือได้อิสระขณะเปลี่ยนสีเสียง

8 Faders — ใหม่ใน MODX M (เดิม MODX/MODX+ มี 4)

MODX M มี Fader 8 ตัว ใช้ได้หลายโหมด:

โหมดFader 1-8 ควบคุม
ปกติVolume ของ Part 1-8 แต่ละ Part แยกกัน
OrganDrawbar ของออร์แกนทั้ง 8 แถบ (Footage 16' 8' 4' ฯลฯ)
FM-XLevel ของ Operator 1-8 ในแต่ละ Part
AN-XLevel ของ OSC1, OSC2, OSC3, Noise และ Parameter อื่นๆ
กด [PERFORMANCE (HOME)] → จิ้มที่แถบ Fader Mode ด้านล่างจอ → เลือกโหมด → ป้ายชื่อเหนือ Fader แต่ละตัวเปลี่ยนทันที
Keyboard Hold & Portamento — บนแผงหน้า

กด [HOLD] → คีย์บอร์ดค้างเสียงไว้แม้ปล่อยนิ้ว → ไปปรับ Knob หรือกด Scene ได้โดยไม่ขาด Sustain

กด [PORTAMENTO] → เปิด/ปิด Portamento → เสียง Slide ระหว่างโน้ต เหมาะกับ Lead Synth แนว Vintage

ไฟ LED ของปุ่ม [HOLD] หรือ [PORTAMENTO] ติดสว่าง = เปิดอยู่ / ดับ = ปิด

4. การจัดการ Performance และ Scene

Live Set — ตารางเสียงพร้อมเล่น

กด [LIVE SET] → จอแสดงตาราง 4×4 = 16 ช่องต่อหน้า (มีหลายหน้า) แต่ละช่องคือ Performance ที่จัดเตรียมไว้

จอแสดง Grid 16 ช่อง — แต่ละช่องมีชื่อ Performance + สีที่กำหนดเอง สลับหน้าด้วยการ Swipe หรือกด Display Knob
จิ้มช่องที่ต้องการ → Performance โหลดทันที ไม่มี Delay ระหว่างเปลี่ยน
กด [LIVE SET] ค้าง → เข้าโหมด Edit Live Set → Drag & Drop เรียงลำดับช่องได้ หรือจิ้มช่องว่าง → Assign Performance ใหม่
Category Search — ค้นหาเสียงขั้นสูง

กด [CATEGORY] ขณะหน้า Performance เปิดอยู่

จอแสดง Category Search: ด้านซ้ายมี Filter (Main/Sub Category + Attribute): Piano, Organ, Strings, AWM2, FM-X, AN-X, Single, Multi ฯลฯ / ด้านขวาแสดงรายการ Performance ที่ตรงกัน
จิ้ม Attribute "AN-X" → กรองเฉพาะ Performance ที่มี AN-X Engine → จิ้มชื่อ Performance เพื่อ Preview / กด [ENTER] เพื่อเลือก
จิ้มไอคอน ★ ข้างชื่อ Performance → เพิ่มใน Favorite → กรองด้วย Bank "Favorite" เพื่อเรียกในระหว่าง Live Performance ได้ทันที
Scenes 1-8 — Macro สลับ Section เพลง

แต่ละ Scene บันทึก Snapshot ของ: Part Volume, Arp On/Off, Motion Sequence On/Off, Effect Parameter ของทุก Part พร้อมกัน

กด Scene Button 1-8 → ปุ่มที่กดสว่างขึ้น + ค่าทั้งหมดที่บันทึกไว้ใน Scene นั้นถูก Recall ทันที เช่น Scene 1 = เสียงเบา (Intro), Scene 3 = Full Texture (Chorus)

บันทึก Scene: ตั้งค่า Parameter ต่างๆ ให้อยู่ที่ต้องการ → กด Scene Button ค้าง 1 วินาที

Pop-up "Scene X Stored" แสดงขึ้น 1 วินาทีแล้วหายไปเอง — กดสลับระหว่าง Scene ขณะเล่นสดได้ทันที

5. การตั้งค่าเชิงลึกและการเชื่อมต่อ

Split & Layer — แบ่งและซ้อนเสียง

กด [EDIT] → เลือก Part → จิ้ม Tab "Zone/Key"

จอแสดง Piano Roll แนวนอน มีแถบสีของแต่ละ Part ซ้อนกัน — จิ้มขอบแถบแล้วลากเพื่อกำหนด Note Range (Note Low / Note High)
Part 1 ตั้ง Note Low = E3 → เสียง Piano อยู่แค่ช่วงบน / Part 2 ตั้ง Note High = D#3 → เสียง Bass อยู่แค่ช่วงล่าง = Split สำเร็จ

Layer: ปล่อยให้ทั้ง 2 Part มี Note Range ซ้อนกันครบทั้งแป้น → กดคีย์ใดก็ได้ยิน 2 Part พร้อมกัน

Effects — Insertion & Master

แต่ละ Part มี Insertion Effect A + B (2 ชุด) สำหรับ Process เสียงเฉพาะ Part เช่น Compressor + EQ เฉพาะ Piano

นอกจากนี้มี Master Effects (Reverb, Variation) รวมทุก Part ก่อนออก Output

กด [EDIT] → Tab "Common/Audio" → Tab "Effect" → จอแสดง Routing Diagram: Part → Ins A → Ins B → Master FX → Output
จิ้มช่อง Ins A หรือ Ins B → เลือก Effect Type จาก Library (Compressor, Chorus, Distortion ฯลฯ) → Display Knob ปรับ Parameter ได้ทันที
USB Audio Interface — ไม่ต้องซื้อ Interface แยก

ต่อ USB Cable จาก MODX M เข้า Computer → เครื่องทำหน้าที่เป็น Audio Interface 2in/2out ในตัว

กด [UTILITY] → Tab "General" → "USB I/O" → ตั้ง "USB Audio" เป็น ON → Computer จะเห็น MODX M เป็น Audio Device ทันที

บันทึกเสียง: DAW เปิด → เลือก Input = MODX M → อัด Audio ได้เลย ทั้งเสียง Synth จากตัวเครื่องและเสียงจาก A/D Input (ไมค์/กีตาร์)

ESP (Expanded Softsynth Plugin) — MODX M บน DAW

Yamaha ESP คือ Software Plugin ที่จำลองเสียงทั้งหมดของ MODX M บน Computer ได้แบบ 1:1

  • แต่งเสียงใน DAW → โหลดกลับมาที่ MODX M ได้ทันที
  • ซ้อมที่บ้านด้วย Laptop โดยไม่ต้องมีเครื่องจริง
  • ใช้ในระหว่าง Session เพื่อ Render Audio จาก ESP แทนการ Record จากเครื่องจริง

ดาวน์โหลดฟรีจาก Yamaha Download Center → ติดตั้งใน DAW เป็น VST3 / AU / AAX
