ภาพรวม MODX M คืออะไร

MODX M คือ Music Synthesizer รุ่นใหม่ล่าสุดของซีรีส์ มีคีย์บอร์ดในตัว (M6 = 61 คีย์, M7 = 76 คีย์, M8 = 88 คีย์) เพิ่ม AN-X Virtual Analog Engine บน AWM2 + FM-X ที่มีอยู่แล้ว รวม 3 Sound Engine ในเครื่องเดียว พร้อม USB Audio Interface 2in/2out

MODX M คืออะไร?

Music Synthesizer มีคีย์บอร์ด

MODX M มีคีย์บอร์ดในตัว แบ่งเป็น 3 รุ่นตามขนาด:

รุ่นจำนวนคีย์ประเภทคีย์
MODX M661 คีย์FSB (Balanced Hammer) Semi-weighted
MODX M776 คีย์FSB Semi-weighted
MODX M888 คีย์BHC (Balanced Hammer Compact) Graded Hammer
3 Sound Engines — รวมกันครั้งแรก

AWM2 — Sample-based สูงสุด 128 Elements ต่อ Part, Filter, EG ครบ
FM-X — 8-Operator FM Synthesis, 88 Algorithm
AN-X — Virtual Analog (ใหม่) 3 Oscillators + Noise Generator, Pulse Width, OSC Self Sync, Wave Shaper

AN-X Engine — Virtual Analog คืออะไร?

AN-X จำลองวงจร Analog Synthesizer แบบ Digital อย่างสมบูรณ์ ประกอบด้วย:

  • OSC 1, OSC 2, OSC 3 — แต่ละตัวปรับ Pulse Width, Wave Shaper, OSC Self Sync ได้
  • OSC 3 → Modulate OSC 1/2 ด้วย FM หรือ Ring Modulation ได้
  • Filter 1 + Filter 2 — ต่อ Output ของ Oscillator ได้อิสระ
  • Noise Generator — ใช้สร้างเสียง Breath, Wind, Percussion

ได้เสียงใกล้เคียง Moog, Prophet — ทำ Acid Bassline, Classic Lead, Warm Pad ได้ดีมาก

MODX M เหมาะกับใคร?

Keyboard Player ที่ต้องการเสียงครบสุด

ต้องการ Synthesizer ที่ครอบคลุม AWM2 acoustic sounds + FM-X electric sounds + AN-X analog sounds ในเครื่องเดียว ไม่ต้องต่อหลายเครื่องบนเวที

Studio Producer / Sound Designer

USB Audio 2in/2out ในตัว ต่อ Computer ได้เลยไม่ต้องการ Audio Interface แยก AN-X Engine ช่วยสร้างเสียง Analog Synth ที่ไม่เหมือนใครโดยไม่ต้องซื้อ Hardware Analog

Live Performer ที่ต้องการ Layer หลายเสียง

Performance Mode รวมสูงสุด 8 Parts, Scene Button 8 ปุ่ม สลับ Sound Configuration ทันที, Super Knob หมุน Control หลาย Parameter พร้อมกัน
