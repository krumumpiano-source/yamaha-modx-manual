ภาพรวม MODX คืออะไร

MODXM คือ Synthesizer Module (ไม่มีคีย์บอร์ด) รุ่นสูงสุดของซีรีส์ เพิ่ม AN-X Analog Physical Modeling Engine บน AWM2 + FM-X ที่มีอยู่แล้ว พร้อม USB Audio Interface 2in/2out ในตัว เหมาะสำหรับ Studio Producer และ Keyboard Player ที่มี Master Keyboard อยู่แล้ว

MODXM คืออะไร?

Sound Module

MODXM ไม่มีคีย์บอร์ดในตัว ต้องใช้ร่วมกับ Master Keyboard ส่ง MIDI มาควบคุม หรือใช้ใน Studio เป็น Sound Source ผ่าน DAW

3 Sound Engines

AWM2 (Sample) + FM-X (8-Op FM) + AN-X (Analog Physical Modeling) — ครบทั้ง 3 Engine ที่ไม่มีในรุ่น MODX และ MODX+

AN-X Engine คืออะไร?

AN-X = Virtual Analog Synthesis จำลองวงจร Analog Synthesizer (Filter, Oscillator) ด้วย Physics Model เสียงที่ได้ใกล้เคียงกับ Analog Synth จริงๆ เช่น Moog, Prophet — ทำเสียง Acid Bassline, Classic Lead, Warm Pad ได้ดีมาก

MODXM เหมาะกับใคร?

Studio Producer

มี Keyboard อยู่แล้ว (ไม่ว่าจะเป็น Controller ก็ดี) ต้องการ Sound Module คุณภาพสูงสุดใน MODX Series

Live Setup ที่ซับซ้อน

วางบน Rack หรือ Desktop ไม่ใช้พื้นที่มาก ต่อ Master Keyboard เล่นสด ประหยัดน้ำหนักในการเดินทาง
