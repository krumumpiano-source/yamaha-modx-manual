วิธีแต่งเสียง AWM2 & FM-X

การแต่งเสียงบน MODX ทำบนหน้าจอ Touch Screen ทั้งหมด — อธิบายทีละขั้นว่ากดอะไร จอขึ้นหน้าไหน จิ้มตรงไหน ปรับค่าอย่างไร สำหรับทั้ง AWM2 และ FM-X

ก่อนเริ่ม — เข้าหน้า Part Edit

ขั้นที่ 1 — เลือก Performance ที่จะแก้

กด [HOME] → จอแสดงหน้าหลัก Performance ปัจจุบัน

ถ้าต้องการเปลี่ยน Performance ก่อน: จิ้มชื่อ Performance ที่บนสุดของจอ → หน้า Category Search เปิดขึ้น → เลือก Performance → จิ้ม OK
ขั้นที่ 2 — เข้า Edit Mode

กด [EDIT] → จอเปลี่ยนเป็นหน้า Performance Edit

บนสุด: Tab "Common/Audio" ไฮไลต์ / ถัดมา: Tab Part 1 ถึง Part 8 (แต่ละ Tab มีสี: ส้ม=AWM2, น้ำเงิน=FM-X)
ขั้นที่ 3 — เลือก Part ที่จะแก้
จิ้ม Tab "Part 1" (หรือ Part ที่ต้องการ) → จอแสดงหน้า Part Edit ของ Part นั้น
จอแสดง Tab แถวบนใหม่: Element / OSC Tune / Filter / Amp / Mod/Control / Effect / Arpeggio

ถ้า Part เป็น FM-X จะเห็น Tab: Algorithm / Op1-Op8 / LFO / Tune / Mod/Control / Effect

แต่งเสียง AWM2 — Tab ทีละ Tab

Tab: OSC Tune (เลือก Waveform)

จิ้ม Tab "OSC Tune"
จอแสดง: Wave Category, Wave Number, Wave Name (ชื่อ Sample เช่น "CFX Grand L"), Octave (-2 ถึง +2), Coarse Tune, Fine Tune, Note Range Low/High
จิ้มที่ "Wave Name" (ชื่อ Sample ปัจจุบัน) → Pop-up Wave Selection เปิดขึ้น → เห็น List ของ Waveform ทั้งหมดในระบบ

หมุน Data Dial เลื่อนดู Waveform → เล่นคีย์ทดสอบเสียงได้ระหว่างเลือก → จิ้ม OK เพื่อยืนยัน

จิ้มค่า "Note Range Low" → ค่าไฮไลต์ → หมุน Dial หรือกด [INC]/[DEC] เพื่อปรับว่าเสียงนี้เริ่มเล่นจากคีย์ไหน (ใช้สำหรับ Split)

Tab: Filter (ปรับความสว่าง/ความมืดของเสียง)

จิ้ม Tab "Filter"
จอแสดง: Filter Type Selector (LPF24, LPF18, HPF ฯลฯ), Slider: Cutoff Frequency, Resonance / ด้านล่าง: Filter EG Graphic — เส้นกราฟ Envelope มี 4 จุดที่ลากได้ (Attack, Decay, Sustain, Release)
จิ้มที่ Slider "Cutoff" แล้วลากขึ้น-ลง (หรือหมุน Dial หลังจิ้ม) → เสียงจะ Bright ขึ้น (Cutoff สูง) หรือ Muffled/Dark ลง (Cutoff ต่ำ)
จิ้มที่ Slider "Resonance" → เพิ่ม Resonance = เสียง "นาสิก/Wah" รอบ Cutoff Frequency เพิ่มขึ้น ลดก็เสียงนุ่มลง
จิ้ม Handle จุด A (Attack) ใน EG Graph แล้วลากซ้าย-ขวา → ควบคุมว่า Filter จะ Open ช้าหรือเร็ว / จุด D (Decay) ควบคุมว่า Filter จะปิดลงหลัง Attack เร็วแค่ไหน

Tab: Amp (ปรับ Volume + Envelope ADSR)

จิ้ม Tab "Amp"
จอแสดง: Level Slider, Pan Knob (ซ้าย-ขวา), Amp EG Graphic — เส้นกราฟ ADSR ลากได้ 4 จุด, Velocity Sensitivity (ความไวต่อแรงกด)
จิ้ม Handle จุด A (Attack) ใน Envelope Graph → ลากขวา = เสียงค่อยๆ ขึ้น ช้า (Pad) / ลากซ้าย = เสียงขึ้นทันที (Piano, Pluck)
จิ้ม Handle จุด D (Decay) → ลากขวา = กดคีย์ค้างแล้วเสียงค่อยๆ ลดลงช้า / ลากซ้าย = ลดเร็ว
จิ้มที่ Sustain Level (เส้นนอน) → ลากขึ้น = เสียงค้าง Level สูง / ลากลง = เสียงเงียบลงหลัง Decay
จิ้ม Handle จุด R (Release) → ลากขวา = เสียงยาวหลังปล่อยคีย์ (Hall Reverb Tail) / ลากซ้าย = ตัดเสียงสั้น
จิ้ม Pan Knob → หมุน Dial ซ้าย = เสียงออกซ้าย / ขวา = เสียงออกขวา / 0 = กลาง

Tab: Mod/Control (Assign Modulation)

จิ้ม Tab "Mod/Control"
จอแสดง: Assign Matrix ตาราง Source → Destination / บนสุดเป็น Mod Wheel Section, ถัดมา Super Knob Section, ล่างสุด Assignable Knob
จิ้มช่อง "Destination" ที่ต้องการแก้ → Pop-up List: เลือก Destination เช่น Filter Cutoff, Resonance, Pitch, LFO Depth ฯลฯ
จิ้มช่อง "Depth" → หมุน Dial ตั้งค่าได้ตั้งแต่ -64 ถึง +63 → ค่า Negative = Mod Wheel ลดค่า, Positive = เพิ่มค่า

แต่งเสียง FM-X — ทีละขั้น

Tab: Algorithm (เลือก Routing ของ Operator)

จิ้ม Tab "Algorithm"
จอแสดง: ไดอะแกรม Flowchart Algorithm ปัจจุบัน — เห็น Box Op1-Op8 เชื่อมกันด้วยลูกศร (Carrier ต่อออก Output, Modulator ต่อเข้า Carrier) / มุมขวาล่างมีตัวเลข Algorithm ปัจจุบัน
จิ้มปุ่ม [<] หรือ [>] ใต้ไดอะแกรม → Algorithm เปลี่ยน → ไดอะแกรมอัปเดตทันที → เล่นคีย์ทดสอบ
จิ้ม Operator Box ใดก็ได้ในไดอะแกรม → Box นั้น Highlight สว่าง → ด้านล่างแสดง Level + Ratio ของ Op นั้น สามารถปรับได้ทันที

Tab: Op1 (ปรับ Operator ทีละตัว)

จิ้ม Tab "Op1" (หรือ Op2-Op8)
จอแสดง: Level Slider (0-99), Frequency Ratio (0.50 ถึง 32.00), Detune (-7 ถึง +7), Velocity Sensitivity / ด้านล่าง: Envelope Graph ของ Op นี้มี จุด L0/L1/L2/L3/L4 + R1/R2/R3/R4 ลากได้
จิ้ม Level Slider แล้วลากขึ้น → ถ้า Op นี้เป็น Carrier: เสียงดังขึ้น / ถ้าเป็น Modulator: เสียง Brighter/More Metallic ขึ้น
จิ้มค่า Frequency Ratio → หมุน Dial → เปลี่ยน pitch relationship ของ Op (1.00 = Unison ของ Carrier, 2.00 = Octave ขึ้น, 0.50 = Octave ลง, 7.00 = เสียง Metallic Bell)
จิ้มจุด R1 (Attack Rate) ใน Envelope → ลากซ้าย = เสียง Attack เร็ว / ลากขวา = ช้า
💡 เคล็ดลับ: Copy Parameter

กด [SHIFT] แล้วจิ้ม Parameter ใดก็ตามในหน้า Edit → Pop-up "Copy / Paste / Initialize" ปรากฏ → จิ้ม Copy → ไปที่ Operator/Element อื่น → [SHIFT] + จิ้ม → จิ้ม Paste — ประหยัดเวลาแต่งเสียงมาก!

💡 Compare (A/B)

ขณะแก้ไข Part ใดๆ กด [SHIFT] + [COMPARE] → เสียงสลับกลับเป็น Version ก่อนแก้ ("ต้นฉบับ") เพื่อเปรียบเทียบ → กดซ้ำเพื่อกลับ Version ที่แก้แล้ว
