แนะนำเมนูทุกเมนู

ไล่เมนูทีละหน้า ตามหน้าจอจริงของ MODX เข้าใจ Structure แล้วจะหาทุกอย่างเจอ

โครงสร้างเมนูหลัก

HOME Screen

หน้าแรกที่เห็นเมื่อเปิดเครื่อง แสดง Performance Name, Part List, Super Knob, Scene — กด [HOME] ตลอดเวลาเพื่อกลับมา

Performance Play

โหมดเล่น Performance ปกติ กด Part เพื่อ Select, กด Edit Zone เพื่อเข้า Edit

Performance Edit [EDIT]

แก้ไข Performance ระดับ Part — Common/Audio, Part Setting, Part Edit (AWM2 or FM-X)

Search [CATEGORY SEARCH]

ค้นหา Voice ตาม Category เช่น Piano, Strings, Synth Lead, FM Electric Piano ฯลฯ

Utility [UTILITY]

ตั้งค่าทั่วไป: Master Tuning, MIDI Channel, USB Mode, Factory Reset, Firmware Update

เมนู Part Edit — AWM2

Tabใช้ทำอะไร
Elementแก้ไขแต่ละ Element (สูงสุด 128 Elements)
OSC/Tuneเลือก Waveform, Tune, Detune, Note Range
FilterLow Pass Filter, Cutoff, Resonance
AmpVolume, Pan, Envelope (ADSR)
Mod/ControlLFO, Super Knob Assign, Mod Wheel
EffectInsertion FX, EQ per Element

เมนู Part Edit — FM-X

Tabใช้ทำอะไร
Operatorเลือก Operator (1-8), ปรับ Level, Frequency Ratio
Algorithmเลือก Algorithm 1-88 ว่า Operator ต่อกันแบบไหน
Envelopeปรับ Attack/Decay/Sustain/Release ของแต่ละ Op
LFOModulation LFO per Operator
💡 เคล็ดลับ

กด [SHIFT] + [EDIT] ในระหว่างเล่นเพื่อเข้า Quick Edit — เปลี่ยน Filter Cutoff / Resonance / EQ โดยตรงจาก 8 Knob บนตัวเครื่อง
