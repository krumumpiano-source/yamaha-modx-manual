ต่อ DAW & MIDI Setup

MODX ต่อ DAW ผ่าน USB MIDI (ไม่มี USB Audio ในตัว) หรือใช้ Audio Interface แยก บทนี้ครอบคลุมทั้งสองแบบ

วิธีต่อ MODX กับ DAW ผ่าน USB MIDI

ขั้นที่ 1

ต่อ USB-B Cable จาก MODX → Computer (USB-A หรือ USB-C ด้วย Adapter)

ขั้นที่ 2

ติดตั้ง Yamaha Steinberg USB Driver จาก yamaha.com/en/products/musical_instruments/synthesizers/modx/ → ดาวน์โหลด Driver ตาม OS

ขั้นที่ 3

กด [UTILITY] บน MODX → Settings → MIDI I/O → ตั้ง MIDI IN/OUT = USB

ขั้นที่ 4

เปิด DAW (Cubase, Ableton, Logic) → เพิ่ม MIDI Track → เลือก Input = MODX MIDI Port 1

ขั้นที่ 5

กด Record บน DAW → เล่นคีย์บน MODX → MIDI Note จะ Record เข้า DAW

ต่อ Audio จาก MODX เข้า DAW (ผ่าน Audio Interface แยก)

ขั้นที่ 1

ต่อ Output L+R ของ MODX (6.3mm TRS) → Input ของ Audio Interface (เช่น Focusrite Scarlett, Yamaha AG)

ขั้นที่ 2

ใน DAW เพิ่ม Audio Track → Input = Input 1/2 ของ Audio Interface → Arm Record → เล่น = Audio Record

ขั้นที่ 3

MIDI Track ใช้ส่ง MIDI กลับมาที่ MODX เพื่อ Playback ผ่าน Sound Engine ของ MODX (ไม่ต้อง Render Audio ทุกครั้ง)

💡 ต้องการ USB Audio?

ถ้าไม่อยากซื้อ Audio Interface แยก ให้ Upgrade เป็น MODX+ ซึ่งมี USB Audio Interface 2in/2out ในตัว ต่อ USB เดียวก็ได้ทั้ง MIDI และ Audio
